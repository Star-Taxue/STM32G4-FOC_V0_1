# 教程：三相逆变桥与栅极驱动

> 面向电机驱动新手，从零理解 FOC 的功率级硬件。

---

## 1. 整体结构：MCU 如何控制电机？

```
电池 (VBAT) ──→ 三相逆变桥 ──→ 电机三相绕组 (A/B/C)
                    ↑
               栅极驱动 IC
                    ↑
           MCU 输出 6 路 PWM (TIM1/TIM8)
```

MCU 不直接驱动 MOSFET——它输出的是 3.3V 弱信号，需要 **栅极驱动** 把信号升压到 10–15V，才能打开功率 MOSFET。**逆变桥** 负责把直流电变成三相交流电，驱动电机旋转。

---

## 2. 三相逆变桥：6 颗 MOSFET 的拓扑

每一相绕组由一个 **半桥** 驱动，三相共 3 个半桥，合计 6 颗 N-MOSFET：

```
        VBAT
         │
    ┌────┴────┐
    │  上管 A  │  (高侧 MOSFET)
    ├─────────┤──────→ 电机 A 相
    │  下管 A  │  (低侧 MOSFET)
    └────┬────┘
         │
        GND
```

三个半桥并接在同一 VBAT 和 GND 之间，输出 A/B/C 三相到电机。

### 为什么全是 N 沟道？
N-MOSFET 导通电阻 Rds(on) 同样规格下远低于 P-MOSFET（省电、少发热）。但 N-MOSFET 需要 **栅极电压高于源极电压** 才能导通——低侧下管好办（源极接地），高侧上管源极电压接近 VBAT，需要特殊驱动（自举电路）。

---

## 3. 半桥的三个状态

| 状态 | 上管 | 下管 | 输出 (Vout) |
|------|------|------|-------------|
| **上拉** | ON | OFF | ≈ VBAT |
| **下拉** | OFF | ON | ≈ GND |
| **禁止** | ON | ON | **短路！烧毁！** |

### 死区时间 (Dead Time)
上下管切换时，MOSFET 不是瞬间关断的（有几十 ns 的关断延迟）。如果上管还没完全关断就让下管导通，上下管直通 → 瞬间大电流 → 烧毁 MOSFET。

**解决方案：** 上管关断后，等一小段时间（死区）再开下管。STM32G431 高级定时器可硬件配置死区，典型值 100–500ns。

```
上管: ──████████──────                ──████████──────
下管: ──────████████──      而不是    ──████████──────  (直通！)
           ↑
         死区 200ns
```

---

## 4. 栅极驱动：为什么要它？

### MCU 引脚直接驱动行吗？
**不行。** 三个原因：

1. **电压不够：** MCU 输出 3.3V，而 MOSFET 栅极需要 10–15V 才能充分导通（Vgs(th) 一般 2–4V，但 Rds(on) 达到标称值需要 Vgs > 10V）
2. **驱动电流不够：** MOSFET 栅极等效一个小电容（Ciss，约 1–5nF），开关瞬间需要 1–2A 的脉冲电流快速充放电。MCU 引脚只能输出 ~20mA，导致开关太慢，MOSFET 在放大区停留过久，发热烧毁
3. **高侧上管无法驱动：** 上管源极不接地（悬在 VBAT），需要 **浮地驱动**

### 栅极驱动 IC 干了什么？
- 把 3.3V 逻辑信号升压到 10–15V 栅极驱动电压
- 提供 A 级脉冲电流，让 MOSFET 快速开关
- 通过 **自举电路 (Bootstrap)** 解决上管浮地驱动问题

---

## 5. 自举电路：高侧驱动的核心

```
             VBAT
              │
          ┌───┴──── 二极管 (阳极接 12V)
          │         │
  12V ────┤      ┌──┴──┬────┤───→ 接半桥输出 (S 点)
  (栅极电源)     │ 自举 │
          │     │ 电容 │
    低侧驱动 ←──┴─────┴────→ 上管栅极驱动
```

**原理分 4 步：**
1. 下管导通时，半桥输出被拉到 GND（≈0V）
2. 自举电容通过二极管被 12V 充到 ~11.5V（二极管压降）
3. 下管关断后，半桥输出浮空
4. 需要开上管时，自举电容上的 11.5V 作为上管栅极的"浮动电源"——栅极电压 = 半桥输出电压 + 电容电压，刚好满足 Vgs > 10V

> 自举电容只能在 PWM 频率足够（>1kHz）时持续充电。如果长时间不开关（如 100% 占空比），电容会放电。需要周期性关断上管让下管打开给电容补电，或换用隔离电源。

**典型自举电容:** 1–10μF 陶瓷电容，耐压 > 25V

## 6. FD6288 举例

FD6288 是一款常用三路半桥栅极驱动 IC，支持 3.3V 逻辑输入、内部死区、自举电路。

### 内部框图（一相）
```
IN → 逻辑处理(施密特) → 电平移位 → 高端浮动驱动 → HO (接上管栅极)
                        ↘ 死区控制 → 低端驱动 → LO (接下管栅极)
```

### 典型外围
```
FD6288 (一相)
        ┌─── VCC (12V)
        │
   VB ├─┤├─ 自举电容(2.2μF) ──┤
   HO ├─────────────── 上管栅极(串 10Ω)
   VS ├─────────────── 半桥输出(S点) + 自举电容另一端
   LO ├─────────────── 下管栅极(串 10Ω)
   GND├─────────────── GND
        │
   IN ├── MCU PWM 引脚
```

- **栅极串联电阻**（10–30Ω）调节开关速度：电阻越大，开关越慢（EMI 小但发热大），需折中
- FD6288 内置死区 200ns，可省去 MCU 端死区配置（但建议 MCU 端也配一套，双重保险）

## 8. 从 PWM 到电机旋转：SVPWM 直观理解

三路半桥输出 A/B/C 三相电压。通过调节每相上管的导通时间（占空比），可以在三相绕组上产生任意方向的合成电压矢量：

```
V_A = VBAT × duty_A
V_B = VBAT × duty_B
V_C = VBAT × duty_C

合成矢量 = V_A + V_B × e^(j120°) + V_C × e^(j240°)
```

SVPWM 算法就是算出：给定一个目标电压矢量（幅值 + 角度），三路 PWM 的占空比应该是多少。FOC 电流环输出 Vd/Vq，经逆 Park/Clark 得到 Va/Vb/Vc，再转成三路 PWM 占空比，写入定时器比较寄存器。

> 这些数学变换在 FOC 教程中详述，本节只需知道：**MCU 算占空比 → 定时器出 PWM → 栅极驱动放大 → MOSFET 功率输出 → 电机转。**

## 9. 新手常见坑

| 坑 | 现象 | 原因 | 解决 |
|----|------|------|------|
| **上下管直通** | 上电瞬间烧管 | 死区不够或没配 | TIM 配置死区 200–500ns |
| **MOSFET 发烫** | 小电流就烫 | Vgs 不够或开关太慢 | 栅极电压 > 10V，栅极电阻调小 |
| **电机不转、抖动** | 上电后电机咔咔响 | 一相不导通或 PWM 相序错 | 万用表测三相是否有波形 |
| **自举电容失效** | 长期高占空比后上管不导通 | 电容放电了 | 占空比上限 < 95%，或用隔离驱动 |
