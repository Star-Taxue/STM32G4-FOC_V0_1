# 电流采样与电压采样 — 原理、选型、电路设计

>  3S–4S 电池，LD2808 电机，双路 FOC，**INA240A2-PWR** 电流采样 + 电阻分压母线采样。

---

##  1. 为什么 FOC 必须精确采电流

FOC（磁场定向控制）的核心是用软件在 MCU 里构造一个虚拟的旋转坐标系（d-q 轴），把三相电流分解为**励磁分量（Id）**和**转矩分量（Iq）**，分别闭环控制。

```
相电流 Ia, Ib, Ic  ──[Clark]──→ Iα, Iβ  ──[Park]──→ Id, Iq
                                                      │
                                                  PI 控制器
                                                      │
                                                ──[反Park]──→ Va, Vb, Vc
```

**采不到电流，这整条链路全断。**

更关键的是：**FOC 不是在任意时刻都能采到有效电流的。** 下面解释这个问题。

---

##  2. 为什么两电阻方案是这个项目的最佳选择

###  2.1 三种电流采样架构

| 方案 | 电阻数 | 原理 | 优点 | 缺点 |
|------|:---:|------|------|------|
| **三电阻** | 3 | 每相一个，分别采 Ia, Ib, Ic | 全时刻可采 | 多一个运放 |
| **两电阻** | 2 | 采 Ia, Ib，Ic = -(Ia+Ib) | 省一路 | 只在特定时刻可采 |
| **单电阻** | 1 | 采母线电流，分时解三相 | 最省 | 软件极其复杂 |

###  2.2 两电阻方案的"可采时刻"问题

两电阻接在下管和地之间（低侧采样），只有在**下管导通时**才有电流流过采样电阻：

```
一相半桥:
        VBAT
         │
    ┌────┴────┐
    │  上管    │ ← 上管导通时，A相电流从VBAT走，不经过采样电阻
    ├─────────┤────── 电机 A 相
    │  下管    │
    └────┬────┘
         │
      [采样电阻]
         │
        GND
```

当上管导通（PWM=高）、下管关闭时，这相的采样电阻里**没有电流**，采出来是 0。

**什么时候两相下管同时导通？** 在 PWM 周期的**中间点**（中心对齐 PWM）：

```
PWM 周期 ──────────────────────────────────────────
             ┌─────┐          ┌─────┐
A相上管: ────┘     └──────────┘     └────
                   ↑                    ↑
             ┌───────────┐    ┌───────────┐
B相上管: ────┘           └────┘           └─
                   ↑                    ↑
             ┌────────────────┐
C相上管: ────┘                └───────────────
                   ↑
              PWM 中心点（三路下管几乎同时导通）
```

在 PWM 中心点，三个下管几乎同时导通 → 两相采样电阻都有电流 → **此时 ADC 触发采样。**

这就是为什么 CubeMX 里 TIM1/TIM8 的 TRGO 要配置为在 PWM 中点触发 ADC。

###  2.3 为什么选低侧（Low-Side）

| | 低侧（本项目） | 高侧（High-Side） | 同相（Inline） |
|---|---|---|---|
| 运放共模电压 | 0V（GND） | VBAT（12V） | 0V ↔ VBAT 跳变 |
| 运放要求 | 普通运放即可 | 需高共模 | 需极高 CMRR + 快速恢复 |
| 采样时刻限制 | **有**（下管导通时） | 无 | 无 |
| 成本 | 低 | 高 | 最高 |

本项目选低侧两电阻，因为：
1. **INA240 虽然能扛高共模（-4V~80V）**，但低侧配置让共模电压恒为 0V，运放工作在最佳工况
2. 两电阻方案省元件（双路电机共 4 颗 INA240 vs 6 颗）
3. "下管导通才可采"的限制由 MCU 定时器同步触发 ADC 解决——这是 G431 硬件能力范围的事

##  3. 采样电阻选型计算

###  3.1 已知条件

| 参数 | 值 | 来源 |
|------|-----|------|
| 电机额定电流 | ~10A | LD2808-700KV |
| 电机峰值电流 | ~20A | 堵转/加速 |
| INA240A2 增益 | **50V/V** | 固定 |
| INA240 供电电压 | 3.3V | MCU 同一电源 |
| ADC 参考电压 | 3.3V | STM32G431 VREF |
| 测量方向 | **双向** | FOC 需要正/负电流 |

###  3.2 双向测量的输出电压范围

双向测量时，0A 输出设在半电源电压（1.65V），正电流输出走高，负电流输出走低：

```
Vout = 1.65V + (I × Rshunt × 50)

 Vout
  ▲
3.3V ┤ ╱
     ┤╱
1.65V┼──── 0A
   ╲  ┤
 0V  └──────────→ I
  -Imax   0   +Imax
```

**可用输出摆幅**：从 1.65V 向两端各约 1.6V（运放输出不能完全到轨，留 50mV 余量）。

```
Vshunt_max（单方向） = 1.6V / 50 = 32mV
```

###  3.3 按电流算采样电阻

| 目标电流 | R = 32mV / I | 标准值 | 满量程实际 |
|----------|-------------|--------|-----------|
| ±15A | 2.13mΩ | **2mΩ** | ±16A |
| ±20A | 1.60mΩ | **2mΩ** | ±16A（20A 时轻微削顶）|
| ±10A | 3.20mΩ | **3mΩ** | ±10.7A |

**推荐：2mΩ**

理由：
- LD2808 在 3S–4S 下持续电流约 8–12A，峰值 15–20A
- ±16A 范围覆盖 80% 工况，极端 20A 时仅轻微饱和（软件可检测并处理）
- 2mΩ 的功率损耗可控

###  3.4 功率核算

```
P = I² × R

持续 10A: 10² × 0.002 = 0.200W
额定 15A: 15² × 0.002 = 0.450W
峰值 20A: 20² × 0.002 = 0.800W
```

**选 1.5W 或 2W、2512 封装**，功率降额 2×，安全。

###  3.5 信号链精度粗估

```
10A 时：Vshunt = 10A × 2mΩ = 20mV
         Vout = 1.65V + 20mV × 50 = 1.65V + 1.0V = 2.65V

INA240 输入失调 25μV → 对应电流误差 = 25μV / 2mΩ = 12.5mA （可忽略）
G431 ADC 12bit / 3.3V → 1 LSB = 0.8mV → 对应电流 = 0.8mV / (2mΩ × 50) = 8mA

总精度主要由 12bit ADC 和采样电阻 1% 公差决定，运放贡献极小。
```

##  4. INA240A2-PWR — 为什么选它

###  4.1 核心参数

| 参数 | 值 | 意味着什么 |
|------|-----|-----------|
| 增益 | 50V/V (A2) | 业内精度最高的固定增益档 |
| 供电 | 2.7–5.5V | 3.3V 直接供，与 MCU 共电源 |
| 共模范围 | **-4V ~ +80V** | 低侧（0V）甚至负压飞轮都能工作 |
| 输入失调 | **±25μV max** | 对应 12.5mA 误差，可忽略 |
| 温漂 | 250nV/°C | 几乎零温漂 |
| 增益误差 | 0.20% max | 比采样电阻 1% 公差小一个数量级 |
| CMRR DC | 132dB | 共模抑制比极高 |
| CMRR AC @50kHz | **93dB** | 关键：抑制 PWM 开关噪声 |
| 带宽 | 400kHz (A2) | 20kHz PWM 下信号不失真 |
| 摆率 | 2V/μs | 响应快，跟随电流变化 |
| 静态电流 | 2.4mA max | 4 颗共 <10mA |
| 封装 | TSSOP-8 (PWR) | 手焊友好 |
| 价格 | ~¥8–12 | 性能对得起价格 |

###  4.2 "增强型 PWM 抑制" 到底是什么

普通运放在 PWM 电机驱动环境下会出问题——不是因为信号本身，而是因为**共模瞬变**：

```
PWM 跳变 (dv/dt 可达 1V/ns)
         │
         ▼
  ┌──────────────────┐
  │    电机相线      │ ← 这里电压剧烈跳变 0V ↔ VBAT
  │                  │
  │  [采样电阻]      │
  │       │          │
  │   ┌───┴───┐      │
  │   │ 运放  │      │ ← 运放输入端看到的是 "小差分信号 + 巨大共模跳变"
  │   └───────┘      │
  └──────────────────┘
```

普通运放遇到这种瞬变，输出会"愣住"几微秒（饱和恢复）或输出尖峰脉冲。INA240 的增强型 PWM 抑制专门解决了这个问题——它内部的信号链设计让它在 PWM 边沿后极快恢复，不会产生虚假输出。

在 FOC 里这意味着：**你在 PWM 中心点采到的电流是真实的，不是被 PWM 噪声污染的数据。**

###  4.3 引脚与功能

```
INA240A2 (TSSOP-8, 顶视图)

        ┌────U────┐
   IN+  │1      8│  VS (电源 3.3V)
   IN-  │2      7│  OUT (输出到 ADC)
   GND  │3      6│  REF1
   NC   │4      5│  REF2
        └─────────┘
```

| 引脚 | 功能 | 接法 |
|------|------|------|
| IN+ | 同相输入 | 接采样电阻"高端"（靠电机侧） |
| IN- | 反相输入 | 接采样电阻"低端"（靠 GND 侧） |
| VS | 电源 | 3.3V + 100nF 去耦 |
| GND | 地 | 信号地 |
| OUT | 输出 | 接 MCU ADC 输入引脚 |
| REF1, REF2 | 基准输入 | 双向模式：REF1=3.3V, REF2=GND |
| NC | 不连接 | 悬空 |

###  4.4 REF1/REF2 配置 —— 设定输出零点

INA240 有两个基准引脚，通过不同的接法配置输出模式。

| 模式 | REF1 接 | REF2 接 | 输出公式 | 本项目用？ |
|------|---------|---------|----------|:---:|
| **双向** | VS (3.3V) | GND | Vout = 1.65V + Vsense×50 | ✅ |
| 单向（低→高） | GND | GND | Vout = 0V + Vsense×50 | ✗ |
| 单向（高→低） | VS | VS | Vout = 3.3V - Vsense×50 | ✗ |
| 外部基准 | Vref | Vref | Vout = Vref + Vsense×50 | ✗ |

我们用双向模式：REF1 接 3.3V，REF2 接 GND，输出零点自动设在 1.65V。

>  **硬件实现**：REF1 和 REF2 直接连到 VS 和 GND 即可，不需要外加电阻分压器。芯片内部内置了 500kΩ 分压网络。

##  5. 完整电路设计

###  5.1 单路电流采样通道（以电机 A 的 A 相为例）

```
                        3.3V
                         │
                        ┌┴┐
                        │ │ 100nF  去耦电容（紧靠 VS 脚）
                        └┬┘
                         │
                    ┌────┴────────────┐
                    │ VS           NC │4 ── 悬空
                    │                 │
 电机A相 ────┐      │ REF1 ──── 3.3V  │
            │      │                 │
       [2mΩ]      │ REF2 ──── GND   │
    (Rshunt)      │                 │
            │      │ IN+ ────┬───────┤
            ├──────┤         │       │
            │      │ IN- ────┼───┐   │
            │      │         │   │   │
  ──────────┘      │ GND ────┴───┴───┤
     GND           │                 │
                   │ OUT ────[Rf]────┼──────→ MCU ADC (PA0)
                   └─────────────────┘

                    图：INA240 单路电流采样
```

###  5.2 输出滤波 —— Rf + Cf

INA240 输出直接进 ADC 前加一级 RC 低通：

```
INA240 OUT ──[Rf 100Ω]──┬──→ ADC 引脚
                        │
                      [Cf 1nF]
                        │
                       GND
```

| 元件 | 值 | 作用 |
|------|-----|------|
| Rf | 100Ω | 隔离运放输出与 ADC 采样电容，防止 ADC 采样瞬间拉低运放输出 |
| Cf | 1nF | 滤除 PWM 频率（20kHz）以上的残留噪声 |

**截止频率**：fc = 1/(2π × 100Ω × 1nF) ≈ 1.6MHz，滤高频噪声但不衰减 20kHz 以内的电流信号。

> 为什么不加大电容做更多滤波？FOC 电流环带宽通常在 1–2kHz，滤波截止太低会引入相位延迟 → 电流环不稳定。1.6MHz 是安全距离。

###  5.3 输入滤波 —— Rf_in + Cf_in（可选，预留）

```
电机侧 ──[Rf_in 10Ω]──┬── IN+
                      │
                    [Cf_in 100pF]
                      │
采样电阻 ──────────────┴── IN- (直连)
```

| 元件 | 值 | 作用 |
|------|-----|------|
| Rf_in | 10Ω | 限流保护、配合 Cf_in 组成差分滤波 |
| Cf_in | 100pF | 差分滤波，削减 PWM 高频毛刺进入运放 |

**截止频率：** fc_diff = 1/(2π × 2×10Ω × 100pF) ≈ 80MHz

>  **预留即可，先不焊。** 如果布板好、走线短，不加也能正常工作。示波器看 IN+/IN- 波形有高频毛刺再加。

##  6. 电压采样 —— 母线电压分压

###  6.1 为什么要采母线电压

FOC 计算 PWM 占空比时需要知道当前母线电压：

```
目标相电压 = 占空比 × VBUS

如果 VBUS 从 12.6V 掉到 9V，而你仍然用旧值算占空比：
→ 实际输出电压偏小 → 电流跟不上 → 转速掉 → 控制崩
```

所以母线电压需要**每个 PWM 周期都采一次**，和电流同步更新。

###  6.2 分压器计算

```
 VBAT ──[R1 33kΩ]──┬── Vout → ADC
                   │
                 [R2 10kΩ]
                   │
                  GND
```

| 参数 | 公式 | 结果 |
|------|------|------|
| 分压比 | R2/(R1+R2) | 10/43 = 0.2326 |
| VBAT=12.6V → Vout | 12.6 × 0.2326 | **2.93V** ✓ (<3.3V) |
| VBAT=16.8V(4S满电) → Vout | 16.8 × 0.2326 | **3.91V** ✗ (>3.3V!) |

**问题：4S 满电 16.8V 时输出 3.91V，超过 ADC 3.3V 参考！**

两种解决方案：

| 方案 | 做法 | 效果 |
|------|------|------|
| **A — 调整分压** | R1=47k, R2=10k | 16.8V → 2.95V ✓ |
| **B — 钳位保护** | R1=33k, R2=10k + BAT54S 钳位 | 超 3.3V 时二极管导通保护 |

**推荐方案 A（调整分压比）：**

```
4S 兼容分压: 47k + 10k, 分压比 = 10/57 = 0.1754

VBAT=12.6V → Vout = 2.21V  （2/3 量程，够用）
VBAT=16.8V → Vout = 2.95V  （安全，不超 3.3V）
VBAT=9V    → Vout = 1.58V  （低电量仍可测）
```

###  6.3 滤波电容

分压中点对地并一个电容：

```
R1 (47k)
  │
  ├──── Vout → ADC
  │
  ├──[Cf 100nF]── GND
  │
R2 (10k)
  │
 GND
```

**Cf 和 R1//R2 构成低通：**

```
R_eq = 47k // 10k ≈ 8.25kΩ
fc = 1/(2π × 8.25kΩ × 100nF) ≈ 193Hz
```

母线电压是低频信号（电池电压变化慢），193Hz 的截止频率远远足够，同时滤掉了 PWM 耦合过来的高频噪声。

>  **关键：** 分压电阻不能太小，否则自己就成负载了。47k+10k 的静态功耗 = 12.6²/57k ≈ 2.8mW，可以忽略。

##  7. 从分流到 ADC —— 完整信号链路

下面以一相电流为例，把信号从进入采样电阻到 MCU 存储为数字量，每一步的物理意义走一遍。

```
电机电流 I_phase
    │
    ▼
┌──────────────┐
│ 采样电阻 2mΩ  │  电流 → 电压: Vshunt = I × 2mΩ
│              │  I=10A → Vshunt=20mV
└──────┬───────┘
       │ 差分 20mV（满量程 ±32mV）
       ▼
┌──────────────┐
│ 输入滤波      │  差分 RC (10Ω + 100pF)，滤 >80MHz 毛刺
│ (可选，预留)  │  对 20kHz PWM 信号本身无影响
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ INA240A2     │  ×50 倍放大 + 中点偏置
│ Gain = 50V/V │  Vshunt=20mV → 1.0V 偏离 1.65V
│ REF = 1.65V  │  Vout = 1.65V + 20mV×50 = 2.65V
└──────┬───────┘
       │ 单端 0.05~3.25V
       ▼
┌──────────────┐
│ 输出滤波      │  RC 100Ω + 1nF, fc≈1.6MHz
│              │  滤 PWM 残留噪声，不衰减 20kHz 内信号
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ MCU ADC      │  12bit, 3.3Vref
│ (G431)       │  2.65V → ADC码 = 2.65/3.3 × 4096 ≈ 3289
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 软件换算      │  I = (ADC码/4096 × 3.3V - 1.65V) / (2mΩ × 50)
│              │  I = (ADC码 - 2048) × 0.008A/LSB
└──────────────┘
```

**换算公式：**

```
I = (ADC_value - 2048) × 0.00805　(A)

推导:
  Vout = ADC_value / 4096 × 3.3
  Vshunt = (Vout - 1.65) / 50
  I = Vshunt / 0.002

  → I = (ADC_value × 3.3/4096 - 1.65) / (50 × 0.002)
      = (ADC_value - 2048) × 3.3 / (4096 × 0.1)
      ≈ (ADC_value - 2048) × 0.00805
```

2048 是理论零点（1.65V / 3.3V × 4096 = 2048）。**实际零点需校准**（见 §9）。

##  8. 布板要点

###  8.1 Kelvin 连接 —— 采样电阻必须四线接法

这是整个电流采样中最容易踩的坑，也是**最不能省的**。

```
 错误接法（走线电阻混入采样）:
 
 ┌─────────[走线 1mΩ]────┬────[走线 1mΩ]───────┐
 │                       │                      │
 │                  [采样电阻 2mΩ]               │
 │                       │                      │
 └────[走线 1mΩ]─────────┴────[走线 1mΩ]───────┘
            ↑                      ↑
          IN+ 测到的是 采样电阻 + 走线电阻 的压降
          → 测出的电流比实际大!

 正确接法（Kelvin 四线）:

 大电流路径: ────────[采样电阻]────────
                      │      │
                  力线路径  感线路径
                      │      │
 电流走力线 ──────────┘      └────→ INA240 IN+
                                          
 另一侧同理:
 大电流路径: ────────[采样电阻]────────
                      │      │
                      │      └────→ INA240 IN-
                      │
 电流走力线 ──────────┘

 感线里几乎没有电流 → 感线上没有压降 → IN+/IN- 测到的就是纯粹的采样电阻压降
```

**直观理解**：把采样电阻当作一个 4 端器件——两个大焊盘走大电流，两个小感应点引出差分信号线。在 PCB 上，从采样电阻 **焊盘内侧** 引出细线到 INA240 的 IN+/IN-。这条细线里几乎没有电流（运放输入阻抗极高），所以线上没有 IR 压降，测到的就是电阻两端真正的电压。

**对应在 PCB 上的做法：**

```
采样电阻 (2512 封装, 顶视图)
   ┌──────────────────┐
   │  大焊盘    大焊盘  │ ← 大电流路径（粗走线, 2mm+）
   │  ╔══════════════╗ │
   │  ║   2mΩ 电阻   ║ │
   │  ╚══════════════╝ │
   │  大焊盘    大焊盘  │
   └──────────────────┘
       ↑  ↑     ↑  ↑
       │  └────────────── 感线 (6mil 细线) → IN+
       │                  
       └───────────────── 感线 (6mil 细线) → IN-

 感线从焊盘侧面/内侧引出，不要从大电流主路径上分叉。
```

###  8.2 INA240 布局要点

| 要点 | 做法 | 原因 |
|------|------|------|
| 去耦电容 | 100nF 紧靠 VS 和 GND 引脚 | 最短回路抑制电源噪声 |
| 输入走线 | IN+/IN- 差分走线，平行、等长 | 差分信号免疫共模噪声 |
| 输入走线长度 | 距采样电阻 < 20mm，尽量短 | 长走线拾取 PWM 辐射 |
| 输出走线 | 单端，经 Rf 后到 ADC | 100Ω 隔离 ADC 电容负载 |
| 远离功率区 | INA240 放在模拟区，远离 MOSFET | 功率开关节点 dv/dt 极大，耦合过来就废了 |
| 地平面 | IN- 接地点和 GND 引脚共地平面 | 单点接地，避免地环路 |

###  8.3 电压采样布局

```
 电机A母线 ──[R1 47k]──┬──[R2 10k]── GND
                       │
                       ├── Vout → ADC
                       │
                      [Cf 100nF]
                       │
                      GND

 要点:
 - R1 靠近 VBAT 输入端，R2 和 Cf 靠近 ADC 引脚
 - GND 使用信号地，不要用功率地（电机大电流流过的地方）
 - 走线远离 PWM 开关节点（TIMx_CHx 到栅极驱动的走线）
```

##  9. 校准

做完硬件后的流程：

###  9.1 零点校准

给电机断电（电流=0），读 ADC 值。理论是 2048，实际因电阻公差、运放失调等会有偏差（通常在 2048 ± 50 内）。

```
// 取 1000 次平均
offset = average_adc_value_when_zero_current

// 后续所有计算用实际 offset:
I = (adc_value - offset) × 0.00805
```

###  9.2 增益校准

使用已知电流源（或串万用表），给电机一相通 1A 恒流，读取 ADC 值，反算实际增益：

```
actual_gain = (adc_value - offset) / (1A / 0.00805)
```

或者在代码里直接用计算值，靠万用表对比验证。对于 10A 量级的应用，2mΩ + 50V/V 的数字化增益精度通常够用。

###  9.3 直流偏置检查

测量 REF1/REF2 实际电压（万用表直接量 INA240 的 REF1 和 REF2 引脚）。

```
预期: REF1 = 3.30V, REF2 = 0V
实际输出零点 = (REF1 + REF2) / 2 = 1.65V
```

##  11. 上电测试

1. **不焊电机，不焊 MOSFET** → 通电测 INA240 各点：
   - VS 引脚：3.3V ± 5%
   - REF1：3.3V，REF2：0V
   - OUT 引脚：约 1.65V（零点，电流=0）

2. **焊上 MOSFET，不接电机** → MCU 输出 50% 占空比 PWM：
   - 示波器看 INA240 OUT 波形，确认 PWM 中心点平坦（无大毛刺）
   - 如果中心点有振铃 → 检查差分走线长度，加大 Cf_in

3. **接电机，限流电源 1A** → 5% 占空比起步：
   - MCU 串口打印 ADC 原始值
   - 手动转电机轴，看采样值有无响应
   - 对比万用表实测电流 vs ADC 计算值

4. **逐步加大电流到额定** → 检查 IN+/IN- 共模电压不超 -4V（低侧采样永远 0V，不会超）

##  12. 常见坑

| 现象 | 可能原因 | 排查 |
|------|----------|------|
| 零点漂移大 | 没焊去耦电容或距离远 | 万用表量 VS，确认 3.3V 干净 |
| ADC 读值全是噪声 | 输出滤波没加或 Cf 虚焊 | 示波器看 OUT 波形 |
| 小电流不准 | 没做零点校准 | 断电读 ADC，减掉偏移 |
| 大电流测不到（饱和） | 采样电阻选大了 | 算 R = 32mV/I_max，检查 2mΩ 是否正确 |
| 电机转起来后 ADC 值乱跳 | PWM 噪声耦合到模拟走线 | 模拟走线远离功率区，查 Kelvin 接法 |
| IN+/IN- 反接 | 正电流变大 OUT 反而变小 | 交换 IN+ 和 IN- |
| REF1/REF2 接错 | 零点不在 1.65V | 对照 §4.4 的表格检查 |

##  13. 为什么不用内置 OPAMP

STM32G431 内置了 4 个运放（OPAMP），理论上可以省掉外置 INA240。但本项目仍然选外置 INA240，原因：

| | G431 内置 OPAMP | 外置 INA240 |
|---|---|---|
| PWM 抑制 | 无 | **增强型，93dB @50kHz** |
| 增益精度 | 需外接反馈电阻（1% × 2） | **内置激光修调，0.2%** |
| 失调电压 | ~1mV（典型） | **25μV max** |
| 共模范围 | 0–VDDA | **-4V ~ +80V** |
| 差分转单端 | 需外接 4 电阻（匹配要求苛刻） | **内置，REF 引脚搞定** |
| PCB 面积 | 4 个运放 + 16 个电阻 | 4 个运放 + 4 个去耦电容 |
| 成本 | "免费" | ¥32（4颗） |

**结论：** 内置 OPAMP 做普通信号放大够用，但做 FOC 电流采样——PWM 噪声环境下需要极高的 CMRR 和快速恢复——外置专用芯片 INA240 远优于自搭电路。¥32 买的是"不用折腾"。
