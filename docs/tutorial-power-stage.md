# 教程：三相逆变桥与栅极驱动 — 完整功率级

> 3S 电池，LD2808 电机，双路 FOC，小体积 PCB，分立单管方案。

---

## 1. 整体结构

```
电池 (3S 9–12.6V) ──→ 三相逆变桥 ──→ 电机 A/B/C 绕组
                          ↑
                     栅极驱动 IC
                          ↑
               MCU 输出 6 路 PWM (TIM1/TIM8)
```

MCU 不直接驱动 MOSFET——输出 3.3V 弱信号，需栅极驱动升压到 10–15V 才能打开功率 MOSFET。

---

## 2. 三相逆变桥：2×6 颗 MOSFET

每一相由 **两颗 N-MOSFET 组成的半桥** 驱动，三相共 3 个半桥：

```
        VBAT
         │
    ┌────┴────┐
    │  上管    │  ← N-MOSFET
    ├─────────┤──────→ 电机 A 相
    │  下管    │  ← N-MOSFET
    └────┬────┘
         │
        GND
```

### 为什么全是 N 沟道？
N-MOS 的 Rds(on) 同样规格下远低于 P-MOS。但 N-MOS 需要 `Vgs > Vgs(th)` 才能导通——下管源极接地好办，上管源极悬在 VBAT 附近，需要 **自举电路** 浮地驱动。

### 单管 vs 双管

| | 分立单管 (2颗/半桥) | SO-8 双管 (1颗/半桥) |
|---|---|---|
| 散热 | 每颗独立热沉 | 两颗共享一个封装 |
| 选型 | 上百款随便挑 | 就几款 |
| 开源界 | Odrive / VESC / SimpleFOC 全部用 | 仅微型板用 |

本项目锁定 **分立单管 AON6354** ×6/路。

---

## 3. 半桥三态与死区

| 状态 | 上管 | 下管 | 结果 |
|------|------|------|------|
| 上拉 | ON | OFF | Vout ≈ VBAT |
| 下拉 | OFF | ON | Vout ≈ GND |
| **禁止** | ON | ON | **短路！直通烧管！** |

### 死区
上下管切换时，MOSFET 有几十 ns 关断延迟。上管还没完全关断就让下管导通 = 上下直通。

```
上管: ████████──────
下管: ──────████████
           ↑ 死区 200ns
```

**STM32G431 高级定时器可硬件配置死区** + **FD6288 内置 200ns 死区** = 双重保险。典型值 200–500ns。

## 4. 栅极驱动

### 为什么 MCU 不能直接驱？
1. **电压不够** — MCU 3.3V，MOSFET Vgs 需 >10V 才达标称 Rds(on)
2. **电流不够** — MOSFET 栅极等效电容 ~2nF，开关瞬间需 A 级脉冲电流。MCU 引脚 20mA → 开关太慢 → 发热
3. **浮地** — 上管源极不接地，需浮地驱动

### FD6288
| 参数 | 值 |
|------|-----|
| 逻辑电平 | 3.3V 兼容 |
| 驱动电流 | 1.5A (峰值) |
| 内置死区 | 200ns |
| 内置自举二极管 | 是 |
| 驱动电压范围 | 7–25V |
| 封装 | SOP-20 |
| 价格 | ~¥3 |

**每路电机一颗，双路共 2 颗。**

### FD6288 一相外围

```
FD6288 (一相)
        ┌── VCC (12V, 接电池或升压)
        │
   VB ├─┤├─ 自举电容 2.2μF ──┤
   HO ├──────────────────── 上管 Gate (串 Rg 4.7Ω)
   VS ├──────────────────── 半桥输出 (MOSFET S/D 节点)
   LO ├──────────────────── 下管 Gate (串 Rg 4.7Ω)
  GND ├──────────────────── GND
        │
   IN ├── MCU PWM 引脚
```

## 5. 自举电路详解

### 问题
上管 N-MOSFET 需要 `Vgs > Vgs(th)` 才能导通。上管导通时，源极电压 ≈ VBAT（12.6V），则栅极需要 >22V。电池只有 12.6V，不够。

### 解决：自举 (Bootstrap)

```
         12V ────►├────┬──── 自举电容 (+) ──── 上管栅极驱动电源
               二极管   │
                       ├──── 自举电容 (-) ──── 半桥输出 (VS)
                       │
                       └──── 上管 Source ──── 下管 Drain
```

**原理 4 步：**

1. **充电阶段**：下管导通 → 半桥输出 (VS) 被拉到 GND ≈ 0V → 12V 通过二极管给自举电容充电到 ~11.5V（减二极管压降）
2. **准备阶段**：下管关断，半桥输出浮空。自举电容保存 11.5V。
3. **上管导通**：上管栅极驱动用自举电容的 11.5V 作为电源 → Vgs = 11.5V → 上管导通
4. **上管导通后**：半桥输出 = VBAT。自举电容负极 = VBAT，正极 = VBAT + 11.5V。二极管反向截止，电容暂时隔离，只给上管栅极供电。

### 自举电容计算

```
C_boot ≥ (Qg + I_qbs × t_on_max) / ΔV_boot

其中：
  Qg = 上管总栅极电荷 (AON6354: 13nC)
  I_qbs = 驱动 IC 静态电流 (FD6288: ~50μA)
  t_on_max = 上管最长导通时间 (20kHz PWM: ~50μs)
  ΔV_boot = 允许的自举电容电压跌落 (取 0.5V)

C_boot ≥ (13nC + 50μA × 50μs) / 0.5V
       = (13nC + 2.5nC) / 0.5V
       = 31nF

→ 2.2μF 远大于 31nF，足够安全，允许低频 PWM 和长导通时间。
```

### 自举电容选型

| 参数 | 值 | 说明 |
|------|-----|------|
| 容量 | **2.2μF** | FD6288 datasheet 推荐 |
| 耐压 | **25V** | 3S 满电 12.6V + 振铃余量 |
| 材质 | **X7R** | 温度特性好，不要 Y5V/Z5U |
| 封装 | **0805** | 好焊 |
| 数量 | 3 颗/路 (每相一颗) | |

### 自举二极管

FD6288 **内置自举二极管**，无需外置。省 6 颗料。

### 占空比限制
上管长时间导通 → 自举电容持续放电无机会充电 → 电容电压跌落 → 上管驱动不足。

```
最大安全占空比 = 1 - (t_charge / T_pwm)

充电所需时间 ≈ 100ns (电容重新充电时间，非常短)
PWM 频率 20kHz, T_pwm = 50μs
→ 最大占空比 ≈ 1 - 0.1/50 = 99.8%，基本不限
```

> 实际受 FD6288 内部 UVLO 限制，建议占空比 <95%。FOC 正常运行时占空比在 10–90% 之间，完全无影响。

## 6. MOSFET 选型 — 锁定 AON6354

### 选型标准

| 参数 | 要求 | AON6354 实值 |
|------|------|-------------|
| Vds | ≥30V | **30V** ✓ |
| Rds(on) @Vgs=10V | ≤5mΩ | **2.6mΩ** ✓ |
| Qg | ≤30nC | **13nC** ✓ |
| Vgs(th) | ≤2V | **1.5V** ✓ |
| 封装 | DFN 小体积 | **DFN5×6** ✓ |
| 淘宝常见度 | 高 | **高** ✓ |

### 损耗 @ 15A 持续

```
导通损耗:  P_cond = I² × Rds(on)
                   = (15A)² × 2.6mΩ × 0.5
                   = 0.29W

开关损耗:  P_sw = 0.5 × Vds × I × (Qg/I_drive) × f_pwm
                 = 0.5 × 12V × 15A × (13nC/1.5A) × 20kHz
                 = 0.08W

单管总损耗 ≈ 0.37W  → 6 颗共 2.2W
20A 峰值: 0.65W/管 → 共 3.9W，DFN5×6 铺铜轻松散热
```

### 封装优势

```
  DFN5×6        SO-8        TO-252
  ┌──────┐     ┌─────┐     ┌─────┐
  │      │     │     │     │     │
  │ 底部 │     │     │     │     │
  │ 铜焊盘│     └─────┘     └─────┘
  └──────┘   5×6mm       10×6.5mm
  5×6mm

DFN5×6 = SO-8 一样大
        + 底部全铜焊盘（热阻远低于 SO-8 的塑料封装 + 引脚导热）
        + 比 TO-252 小 60%
        + 引脚比 DFN8 长，热风枪好焊
```

## 7. 栅极电阻

```
FD6288 HO/LO ──[Rg]─── AON6354 Gate
                      │
                    [10kΩ] ── GND (下拉，防浮空误导通)
```

| Rg | 开关时间 | 效果 |
|----|----------|------|
| 0Ω | ~9ns | 最快，可能振铃 |
| 4.7Ω | ~15ns | 微阻尼，推荐 |
| 10Ω | ~25ns | 偏保守 |
| 22Ω | ~50ns | 太慢 |

**推荐 4.7Ω**。AON6354 Qg=13nC 极小，FD6288 1.5A 驱动游刃有余，4.7Ω 防轻微振铃同时几乎不影响开关速度。

## 10. 上电测试顺序

1. **不焊 MOSFET**，只通电 FD6288：测 VCC=电池电压？自举电容两端有 12V？
2. MCU 输出 50% 占空比 20kHz PWM，示波器看 HO/LO 有波形、上下管不重叠
3. 焊 MOSFET，**电机不接**，万用表测三相之间无短路、对地无短路
4. 接电机，电源限流 **1A**，给 5% 占空比，看电机是否微动
5. 逐步加占空比和电流限值
6. **死区务必先配好再上电**：TIM1/TIM8 配置死区 200ns 起

## 11. 常见坑

| 坑 | 现象 | 原因 | 解决 |
|----|------|------|------|
| 直通烧管 | 上电瞬间冒烟 | 死区没配/不够 | TIM 死区 ≥200ns |
| MOSFET 烫 | 1A 就烫手 | Vgs 不够 | 电池直供驱动，9V 以上没问题 |
| 电机抖动 | 咔咔响不转 | 一相不导通/PWM 相序错 | 示波器逐相查 |
| 自举失效 | 大占空比时上管不导通 | 电容没机会充电 | 占空比 <95% |
| DFN 虚焊 | 时通时不通 | 底部焊盘没焊好 | 加热台 240°C 或热风枪多吹几秒 |
