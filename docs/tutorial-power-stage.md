# 三相逆变桥与栅极驱动 — 原理、选型、原理图

> 3S 电池，LD2808 电机，双路 FOC，分立单管 **AON6354** + 栅极驱动 **FD6288**。

---

## 1. 整体结构

```
电池 (3S 9–12.6V) ──→ 三相逆变桥 ──→ 电机 A/B/C 绕组
                          ↑
                     栅极驱动 IC
                          ↑
               MCU 输出 6 路 PWM (TIM1/TIM8)
```

MCU 输出 3.3V 弱信号 → 栅极驱动升压到 10–15V → 打开功率 MOSFET → 逆变桥输出三相交流。

---

## 2. 三相逆变桥：6 × AON6354

每相由 **两颗 N-MOSFET 组成半桥**，三相共 3 个半桥：

```
        VBAT
         │
    ┌────┴────┐
    │  上管    │  AON6354
    ├─────────┤──────→ 电机 A 相
    │  下管    │  AON6354
    └────┬────┘
         │
        GND
```

### 为什么全是 N 沟道？
N-MOS 同规格 Rds(on) 远低于 P-MOS。但需要 `Vgs > Vgs(th)` 导通——下管源极接地好办，上管源极悬在 VBAT，需 **自举电路** 浮地驱动。

### 单管 vs 双管

| | 分立单管 | SO-8 双管 |
|---|---|---|
| 散热 | 每颗独立热沉 | 两颗共享封装 |
| 选型自由度 | 上百款随便挑 | 就几款 |
| 开源界 | Odrive/VESC/SimpleFOC 全用 | 仅微型板 |

本项目 **分立单管 AON6354 ×6/路**。

---

## 3. 半桥三态与死区

| 状态 | 上管 | 下管 | 结果 |
|------|------|------|------|
| 上拉 | ON | OFF | Vout ≈ VBAT |
| 下拉 | OFF | ON | Vout ≈ GND |
| **禁止** | ON | ON | **直通烧管** |

### 死区
MOSFET 有数十 ns 关断延迟，上下管同时切换时可能短暂直通。

```
上管: ████████──────
下管: ──────████████
           ↑ 死区 200ns
```

**双重保险：** STM32G431 TIM 硬件死区 + FD6288 内置 200ns 死区。典型值 200–500ns。

---

## 4. 栅极驱动 FD6288

### MCU 不能直接驱 MOSFET
1. **电压** — MCU 3.3V，MOSFET Vgs 需 >10V
2. **电流** — 栅极等效电容 ~2nF，开关瞬间需 A 级脉冲，MCU 仅 20mA
3. **浮地** — 上管源极不接地

### FD6288 参数

| 参数 | 值 |
|------|-----|
| 逻辑电平 | 3.3V 兼容 |
| 驱动电流 | 1.5A 峰值 |
| 内置死区 | 200ns |
| 内置自举二极管 | 是（但建议外加肖特基，见 §5） |
| 驱动电压 | 7–25V |
| 封装 | SOP-20 |
| 价格 | ~¥3 |

每路电机一颗，双路共 2 颗。

## 5. 自举电路详解

### 问题
上管导通时源极 ≈ VBAT (12.6V)，栅极需 >22V。电池只有 12.6V。

### 原理

```
         12V ────►├────┬──── 自举电容 (+) ──── 上管栅极驱动电源
               二极管   │
                       ├──── 自举电容 (-) ──── 半桥输出 (VS)
                       │
                       └──── 上管 Source ──── 下管 Drain
```

**4 步：**
1. 下管导通 → VS 拉到 GND → 12V 通过二极管给电容充电到 ~11.5V
2. 下管关断，电容保存 11.5V
3. 需要开上管时，电容的 11.5V 作为上管栅极浮动电源 → Vgs=11.5V
4. 上管导通后 VS=VBAT，电容负极=VBAT，正极=VBAT+11.5V，二极管反向隔离

### 自举电容计算

```
C_boot ≥ (Qg + I_qbs × t_on_max) / ΔV

  Qg = 13nC (AON6354)
  I_qbs ≈ 50μA (FD6288 静态)
  t_on_max ≈ 50μs (20kHz PWM 一周期)
  ΔV = 0.5V (允许压降)

  C ≥ (13nC + 2.5nC) / 0.5V = 31nF

→ 2.2μF 远大于 31nF，安全。
```

### 自举二极管 —— 请外加 SS14

FD6288 内置自举二极管，但开源项目（Odrive、VESC、SimpleFOC）**一律外加肖特基**：

| | FD6288 内置 | 外置 SS14 |
|---|---|---|
| 类型 | IC 内部伪二极管 | 金属-半导体肖特基 |
| Vf | ~0.7–1.0V | **~0.4V** |
| 反向恢复 | 有 | **几乎为零** |
| 充电结果 | 电容充到 ~11V | 电容充到 ~11.6V |
| 成本 | "免费" | ¥0.1/颗 |

> 3S 低电量时电压仅 9V。减 1V 内置压降只剩 8V 栅极驱动。外置肖特基省 0.6V，多出的就是安全裕量。

**推荐 SS14**（SMA, 40V, 1A, Vf≈0.4V, ¥0.1）。
阳极接 VB (12V)，阴极接 VS (半桥输出)，与内置并联。3 颗/路。

### 自举元件汇总

| 元件 | 型号/值 | 数量/路 |
|------|---------|---------|
| 自举电容 | 2.2μF 25V X7R 0805 | 3 |
| 自举二极管 | SS14 (SMA) | 3 |

## 6. MOSFET 选型 — 锁定 AON6354

| 参数 | 要求 | AON6354 |
|------|------|---------|
| Vds | ≥30V | **30V** |
| Rds(on) @10V | ≤5mΩ | **2.6mΩ** |
| Qg | ≤30nC | **13nC** |
| Vgs(th) | ≤2V | **1.5V** |
| 封装 | DFN5×6 | **5×6mm 底部焊盘** |

### 损耗 @ 15A

```
P_cond = (15A)² × 2.6mΩ × 0.5 = 0.29W
P_sw   = 0.5 × 12V × 15A × (13nC/1.5A) × 20kHz = 0.08W
单管总损耗 ≈ 0.37W，6 颗共 2.2W
```

### 封装对比

| | DFN5×6 | SO-8 | TO-252 |
|---|---|---|---|
| 大小 | 5×6mm | 5×6mm | 10×6.5mm |
| 底部焊盘 | 全铜 | 无（塑料） | 全铜 |
| 散热 | ★★★ | ★ | ★★★ |
| 热风枪 | 轻松 | 轻松 | 烙铁也行 |

DFN5×6 = SO-8 的尺寸 + TO-252 的散热能力。

## 7. 原理图绘制

### 7.1 一相半桥完整电路

```
                        VBAT
                         │
                   ┌─────┴─────┐
                   │  上管 Q1   │  AON6354
                   │  D ────   │  Pin 5-8
                   │  S ────   │  Pin 1-3
                   └──┬──┬──┬──┘
                      │  │  │
          ┌───────────┘  │  └─── Gate (Pin 4) ──[4.7Ω]── FD6288 HO
          │              │
          │         ┌────┴────┐
          │         │  SS14   │  自举二极管 (阳极→12V, 阴极→VS)
          │         └────┬────┘
          │              │
          │         ┌────┴────┐
          │         │ 2.2μF   │  自举电容
          │         └────┬────┘
          │              │
          ├── 电机 A 相 ──┼────────── FD6288 VS
          │              │
          │    ┌─────────┴─────────┐
          │    │     下管 Q2       │  AON6354
          │    │     D ────────    │  Pin 5-8
          │    │     S ────────    │  Pin 1-3
          │    └──┬────┬──────────┘
          │       │    │
          │       │    └── Gate (Pin 4) ──[4.7Ω]── FD6288 LO
          │       │
          │  ┌────┴────┐
          │  │ 采样电阻 │  10mΩ (低侧)
          │  └────┬────┘
          │       │
          └───────┴── GND
```

> A/B/C 三相完全一样，复制粘贴替换网络名即可。

### 7.2 AON6354 引脚

```
AON6354 (DFN5×6, 顶视图)
  ┌─────────────────────┐
  │  S  S  S  G         │  Pin 1,2,3 = Source; Pin 4 = Gate
  │                     │
  │   (底部焊盘 = Drain)  │
  │                     │
  │  D  D  D  D         │  Pin 5,6,7,8 = Drain
  └─────────────────────┘

原理图符号:   D ───┤
            G ───┤ AON6354
                  ├─── S
```

### 7.3 FD6288 一相引脚

```
VCC  ─── 12V
GND  ─── 电源地

U相示例 (V/W 相查 datasheet):
  IN_UH ─── MCU TIMx_CH1
  IN_UL ─── MCU TIMx_CH1N (或接地,用内置死区)
  VB_U  ─── 12V ──┤├── VS_U    (2.2μF 自举电容)
                   SS14         (外置二极管,阳极12V,阴极VS_U)
  VS_U  ─── 半桥输出 (Q1_S / Q2_D)
  HO_U  ──[4.7Ω]── Q1 Gate
  LO_U  ──[4.7Ω]── Q2 Gate
```

### 7.4 栅极回路

```
FD6288 HO ──[Rg 4.7Ω]── AON6354 Gate
                            │
                       [Rgs 10kΩ]
                            │
FD6288 VS ───────────── AON6354 Source
```

**Rg 紧贴 MOSFET Gate 脚**，HO-走线和 VS-走线围成环路尽量小 → 减少 EMI 耦合误触发。

### 7.5 三相总览

```
       VBAT ──┬─────────┬─────────┬──────────
              │         │         │
        ┌─────┴──┐  ┌───┴───┐  ┌─┴─────┐
 FD6288 │ Q1_U   │  │ Q1_V  │  │ Q1_W  │  ← 上管 ×3
        │        │  │       │  │       │
        │  ├──A──┤  ├──B───┤  ├──C────┤  ← 三相输出
        │  │ Q2_U │  │ Q2_V  │  │ Q2_W  │  ← 下管 ×3
        └──┴──┬───┴──┴──┬───┴──┴──┬────┘
              │         │         │
             Rsh      Rsh      Rsh    (10mΩ)
              │         │         │
              └─────────┴─────────┴── GND

MCU TIM1/TIM8 ──→ FD6288 IN ──→ HO/LO ──→ 6×AON6354 ──→ 电机
```

### 7.6 布板要点

| 要点 | 做法 |
|------|------|
| 功率回路最短 | VBAT→Q1_D→Q1_S→Q2_D→Q2_S→Rsh→GND 闭环面积最小 |
| 栅极回路短 | Rg 贴 MOSFET Gate，HO/VS 走线并行靠近 |
| 底部焊盘 | AON6354 底部大面积铺铜 + 过孔到背面 |
| 自举元件 | 电容和 SS14 紧靠 FD6288 VB/VS |
| 星形地 | 功率地 + 信号地单点汇合在采样电阻处 |

### 7.7 EDA 操作提示

1. 画一相完整电路 → 复制粘贴出另外两相，改网络标签
2. FD6288 20pin 元件，三路 HO/LO 分别连三相栅极电阻
3. AON6354 如库没有自建：DFN5×6，5pin (G, S, S, D, D)
4. 用网络标签连同名节点，避免原理图一堆交叉线

## 8. 栅极电阻

| Rg | 开关速度 | 效果 |
|----|----------|------|
| 0Ω | ~9ns | 最快，可能振铃 |
| **4.7Ω** | ~15ns | 微阻尼，推荐 |
| 10Ω | ~25ns | 偏保守 |

AON6354 Qg=13nC，FD6288 1.5A 驱动，4.7Ω 防轻微振铃同时几乎不影响速度。

## 11. 上电测试

1. 不焊 MOSFET → 通电测 FD6288 各点电压、自举电容两端
2. MCU 出 50%/20kHz PWM → 示波器看 HO/LO 波形、上下管不重叠
3. 焊 MOSFET，电机不接 → 万用表测三相间无短路
4. 接电机，限流 1A，5% 占空比起步
5. **死区 200ns 先配好再上电**

## 12. 常见坑

| 坑 | 现象 | 原因 | 解决 |
|----|------|------|------|
| 直通烧管 | 上电冒烟 | 死区没配 | TIM 死区≥200ns |
| MOSFET 烫 | 1A 就烫 | Vgs 不够 | 9V 以上没问题 |
| 电机抖动 | 咔咔不转 | 一相不导通 | 示波器逐相查 |
| 自举失效 | 大占空比上管不导通 | 电容放光 | 占空比<95% |
| DFN 虚焊 | 时通时不通 | 底部焊盘没焊好 | 热风枪多吹几秒/加热台 240°C |
