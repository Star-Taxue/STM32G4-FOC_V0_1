# 三相逆变桥与栅极驱动 — 原理、选型、原理图

> 3S 电池，LD2808 电机，双路 FOC，分立单管 **AON6354** + 栅极驱动 **FD6288**。

---

## 1. 整体结构

```
电池 (3S 9–12.6V) ──→ 三相逆变桥 ──→ 电机 A/B/C 绕组
                          ↑
                     栅极驱动 IC
                          ↑
               MCU 输出 6 路 PWM (TIM1/TIM8)
```

MCU 输出 3.3V 弱信号 → 栅极驱动升压到 10–15V → 打开功率 MOSFET → 逆变桥输出三相交流。

---

## 2. 三相逆变桥：6 × AON6354

每相由 **两颗 N-MOSFET 组成半桥**，三相共 3 个半桥：

```
        VBAT
         │
    ┌────┴────┐
    │  上管    │  AON6354
    ├─────────┤──────→ 电机 A 相
    │  下管    │  AON6354
    └────┬────┘
         │
        GND
```

### 为什么全是 N 沟道？
N-MOS 同规格 Rds(on) 远低于 P-MOS。但需要 `Vgs > Vgs(th)` 导通——下管源极接地好办，上管源极悬在 VBAT，需 **自举电路** 浮地驱动。

### 单管 vs 双管

| | 分立单管 | SO-8 双管 |
|---|---|---|
| 散热 | 每颗独立热沉 | 两颗共享封装 |
| 选型自由度 | 上百款随便挑 | 就几款 |
| 开源界 | Odrive/VESC/SimpleFOC 全用 | 仅微型板 |

本项目 **分立单管 AON6354 ×6/路**。

---

## 3. 半桥三态与死区

| 状态 | 上管 | 下管 | 结果 |
|------|------|------|------|
| 上拉 | ON | OFF | Vout ≈ VBAT |
| 下拉 | OFF | ON | Vout ≈ GND |
| **禁止** | ON | ON | **直通烧管** |

### 死区
MOSFET 有数十 ns 关断延迟，上下管同时切换时可能短暂直通。

```
上管: ████████──────
下管: ──────████████
           ↑ 死区 200ns
```

**双重保险：** STM32G431 TIM 硬件死区 + FD6288 内置 200ns 死区。典型值 200–500ns。

---

## 4. 信号流程全景：一路 PWM 到一相输出的完整通路

从 MCU 输出一路 PWM 信号开始，到半桥输出驱动电机，电信号完整流经以下环节。
每经过一个环节，解释需要什么元件、为什么。

```
MCU定时器 ──→ FD6288 ──→ 自举 ──→ Rg ──→ MOSFET Gate ──→ 半桥输出 → 电机
 (3.3V)     (升压整形)   (浮地电源)  (调速)  (开关动作)     (功率输出)
```

> 下面以一相上管从关断到开通的切换过程为例，走一遍完整通路。

### 步骤 1：MCU 输出 PWM (3.3V 弱信号)

MCU 定时器比较寄存器更新 → TIMx_CH1 引脚输出 3.3V 方波。

**此时信号特征：** 3.3V 逻辑电平，最大驱动能力 ~20mA。这里没有任何外加元件——只是 MCU 内部产生的一根信号线。

### 步骤 2：进入栅极驱动 IC (FD6288)

MCU PWM 进入 FD6288 的 IN_UH 引脚。

**FD6288 在这个阶段干三件事：**

| 做了什么 | 为什么需要 |
|----------|-----------|
| **施密特触发整形** | 消除 PWM 边沿噪声，防止逻辑误判 |
| **电平移位 + 放大** | 3.3V 逻辑信号 → 12V 栅极驱动信号（MOSFET 需要 Vgs>10V），驱动电流从 20mA → 1.5A 峰值 |
| **死区插入** | 上下管 IN_UH / IN_UL 即使同时收到信号，输出 HO / LO 会错开 200ns，防止直通 |

**至此信号变成：** 12V、1.5A 驱动能力的栅极信号，但还有一个问题——上管 HO 输出的"地"在哪里？

### 步骤 3：自举电路 —— 解决"上管的 12V 参考哪里"

下管 LO 输出好办：它的"地"就是 GND。LO 输出 12V → 下管栅极对源极为 12V → Vgs=12V ✓。

**上管 HO 的问题：** 上管源极不接地，悬在 VBAT 和 GND 之间。如果 HO 输出参考 GND，那么当上管导通、源极 = VBAT=12.6V 时，栅极 12V 对源极 12.6V，Vgs = -0.6V → **不可能导通**。

**自举解决：** 创造一组"浮动电源"，正极比上管源极高 12V，负极为上管源极本身。

```
自举电路演进 (看懂每一步加了什么、为什么加)：

原始状态:                    → 加自举电容:                → 加外置肖特基:
                                                        
 12V ──── FD6288 VB         12V ──┤├──┐ FD6288 VB      12V ──►├──┬──┤├── FD6288 VB
        (电源输入)            (内置) │     │              SS14   │   │
                                    │     │                     │   │
                              ┌────┴──┐  │               ┌─────┴──┐│
                              │ 2.2μF │  │               │ 2.2μF  ││
                              │ 自举   │  │               │ 自举   ││
                              └───┬───┘  │               └───┬────┘│
                                  │      │                   │     │
                      FD6288 VS ─┘      │        FD6288 VS ─┘     │
                       (连半桥输出)       │         (连半桥输出)    │
                                         │                        │
                         上管栅极驱动 ←──┘           上管栅极 ←───┘
```

**三层结构，逐层解释：**

#### 第一层：仅有自举电容（FD6288 内置二极管充电）

**自举电容的作用——"电池"：** 为什么不是直接把 12V 接到上管栅极驱动？因为那是参考 GND 的固定 12V，而上管需要参考源极的浮动 12V。自举电容就是一个"会移动的电池"——它的负极挂在半桥输出（上管源极），正极输出比负极高 12V 的电压。

**充电时机：** 下管导通时，半桥输出被拉到 GND → 电容负极 = GND → 12V 通过内置二极管给电容充电到 ~11V（减内置压降 1V）。

**放电时机：** 需要开上管时，电容的 11V 作为浮动电源驱动上管。上管导通后电容负极 = VBAT，正极 = VBAT+11V → Vgs=11V，上管完全导通。

**仅用电容够用吗？** 勉强够，但有问题：

| 时刻 | 电池电压 | 内置二极管压降 | 自举电容实际电压 | 栅极驱动电压 Vgs |
|------|---------|:---:|---------------|----------------|
| 满电 | 12.6V | 1V | 11.6V | **11.6V** ✓ |
| 半电 | 11.1V | 1V | 10.1V | **10.1V** ✓ |
| 低电 | 9.0V | 1V | 8.0V | **8.0V** ✗ |

3S 低电量时 Vgs 仅 8V。AON6354 在 Vgs=8V 时 Rds(on) 比 10V 标称值高约 20%，意味着更多发热——而这恰恰发生在电池电量最低、最需要效率的时候。

**这就是为什么要加第二层：外置肖特基 SS14。**

#### 第二层：外置肖特基 SS14

**SS14 的作用——"抢电流的捷径"：** SS14 并联在 12V 和自举电容正极之间，与 FD6288 内置二极管并联。两者同时存在——电流走压降小的一路，肖特基 Vf=0.4V，内置伪二极管 Vf≈1V → 电流主要走 SS14。

**加不加 SS14 的区别：**

| 时刻 | 无 SS14 (仅内置) | 有 SS14 | 收益 |
|------|:---:|:---:|------|
| 满电 12.6V | 11.6V | **12.2V** | 0.6V |
| 低电 9.0V | 8.0V | **8.6V** | 0.6V |

8.6V vs 8.0V 的 0.6V 差异，是 AON6354 发热增加 15% 还是 20% 的分界线。SS14 的成本是 ¥0.1。

**为什么还要保留自举电容？** 电容仍是"会移动的电池"——SS14 是充电的导线，电容是储存电能的容器。没有电容，电荷无处存储，二极管导通时栅极驱动只能得到 12V 的瞬时脉冲，关断时电压立即消失。有了电容，电荷存起来，需要时取用。

#### 第三层：为什么不必再加更多？

至此自举电路完成：SS14（充电捷径） + 2.2μF（储能容器） = 给上管提供稳定可靠的浮动电源。

有人可能问：要不要再并个稳压管？不需要。自举电容上的电压由 12V 电源决定，最高 12.6V，远低于 MOSFET 栅极耐压（通常 ±20V），不会过压。

### 步骤 4：栅极电阻 Rg + 并联二极管（开关速度控制）

FD6288 HO/LO 输出 12V 驱动信号 → 经栅极电阻 Rg → 到达 MOSFET 栅极。

**为什么不在 HO 和 Gate 之间直连（0Ω）？**

MOSFET 栅极等效一个电容（AON6354 Ciss≈2nF）。HO 输出 12V 的瞬间，如果直连，电流只受 HO 输出内阻和 PCB 走线电感限制——电流上升极快（数 A/ns）→ PCB 走线电感 × di/dt = 电压振铃 → 栅极电压可能过冲超过 20V → 损坏栅极氧化层。

**Rg=4.7Ω 的作用——"油门踏板"：**
```
HO ──[4.7Ω]── Gate (Ciss≈2nF)

充电时间 τ = Rg × Ciss = 4.7Ω × 2nF ≈ 9.4ns
开关时间 ≈ 2.2τ ≈ 20ns
```
Rg 限制了对栅极电容的充电电流峰值，让开关速度可控。代价是开关变慢 → 开关损耗略增。4.7Ω 是 "快但不振铃" 的平衡点。

**为什么 Rg 两端并二极管？——"开慢关快"**

开管时可以慢一点（走 Rg，控 EMI），关管时必须快——对面管子马上就要开，这边关慢了就直通烧管。

```
HO ──┬──[Rg 4.7Ω]──┬── Gate
     │              │
     └──►├──────────┘
       (1N4148, 阴极朝 Gate)

开管: HO 输出 12V → 二极管反向截止 → 电流走 Rg → 慢开
关管: HO 输出 0V  → Gate 上 12V 通过二极管正向泄放到 HO → 绕过 Rg → 快关
```

> **本项目：可省略。** AON6354 Qg=13nC 极小，Rg=4.7Ω 关断只要 ~15ns，远快于死区 200ns。如果布板走线短、寄生电感小，不加也安全。预留焊盘即可。

### 步骤 5：栅源电阻 Rgs + 栅源电容 Cgs（栅极保护）

信号到达 MOSFET 栅极后，最后两道保险。

**Rgs=10kΩ —— 下拉电阻**

```
Gate ──[10kΩ]── Source
```

**场景：** MCU 上电前，所有 IO 高阻态，FD6288 未激活，HO 输出不确定。没有 Rgs 时，栅极浮空，一点点漏电流或 PCB 上耦合的噪声可能让栅极电压超过 Vgs(th)=1.5V → MOSFET 微微导通 → 电流流经尚未就绪的电路 → 可能烧管。

10kΩ 提供一条确定的低阻抗路径到地（源极），确保栅极默认状态为低电平。阻值取 10kΩ 是因为：太小（<1kΩ）→ FD6288 HO 输出时持续耗电，太大（>100kΩ）→ 下拉太弱，容易受噪声影响。

**Cgs=220pF —— 防寄生导通**

```
Gate ──┬──[Cgs 220pF]── Source
       │
```

**场景：下管关断、上管开通的瞬间。** 半桥输出电压从 0V 跳变到 VBAT（比如 12V），上升速率 dv/dt 可达数十 V/ns。这个快速跳变通过下管的 **米勒电容（Cgd，栅-漏电容）** 耦合到下管栅极：

```
                    Cgd (~400pF)
  半桥输出 (漏极) ────┤├──── 栅极
                    耦合路径
```

dv/dt 越大，耦合电流越大：`I_couple = Cgd × dv/dt`。这个电流流经 Rgs 产生电压：`V_spike = I_couple × Rgs`。

如果 V_spike > Vgs(th)=1.5V → 下管被"弹开" → 上下管瞬间同时导通 → 直通烧管。

**Cgs 的作用——"分流阀"：** Cgs 和 Cgd 构成电容分压器：
```
V_gate = V_drain × Cgd / (Cgd + Cgs)
       = 12V × 400pF / (400pF + 220pF)
       = 7.7V (无 Cgs 时全耦合)
       = 12V × 400pF / (400pF + 220pF) ≈ 7.7V

不对，这是直流分压。交流 dv/dt 耦合时：
V_spike = I_couple × Z_cgs
        = (Cgd × dv/dt) × (1 / (2πf × Cgs))
加 Cgs 后等效阻抗降低，尖峰被抑制。
```

更直观的理解：Cgs 给耦合过来的电荷提供了一条低阻抗泄放路径，像在栅极对地加了个"大水池"——Cgd 注入的一小杯水，在大水池里掀不起什么浪。

**建议：留焊盘，先不焊。** 布板好、走线短时寄生电感小、dv/dt 耦合弱，可能不需要。先上电，示波器看栅极波形，有超过 1V 的尖峰再加。

### 步骤 6：半桥输出 → 电机

栅极信号打开 MOSFET → 上管导通时电流从 VBAT 经上管流向电机相线，下管导通时电流从电机相线经下管流向 GND。三相 PWM 交替开关 → 在电机绕组上合成旋转磁场 → 转子转动。

**至此，一条 PWM 信号的完整旅程结束。**

## 6. 自举电路详解

### 问题
上管导通时源极 ≈ VBAT (12.6V)，栅极需 >22V。电池只有 12.6V。

### 原理

```
         12V ────►├────┬──── 自举电容 (+) ──── 上管栅极驱动电源
               二极管   │
                       ├──── 自举电容 (-) ──── 半桥输出 (VS)
                       │
                       └──── 上管 Source ──── 下管 Drain
```

**4 步：**
1. 下管导通 → VS 拉到 GND → 12V 通过二极管给电容充电到 ~11.5V
2. 下管关断，电容保存 11.5V
3. 需要开上管时，电容的 11.5V 作为上管栅极浮动电源 → Vgs=11.5V
4. 上管导通后 VS=VBAT，电容负极=VBAT，正极=VBAT+11.5V，二极管反向隔离

### 自举电容计算

```
C_boot ≥ (Qg + I_qbs × t_on_max) / ΔV

  Qg = 13nC (AON6354)
  I_qbs ≈ 50μA (FD6288 静态)
  t_on_max ≈ 50μs (20kHz PWM 一周期)
  ΔV = 0.5V (允许压降)

  C ≥ (13nC + 2.5nC) / 0.5V = 31nF

→ 2.2μF 远大于 31nF，安全。
```

### 自举二极管 —— 请外加 SS14

FD6288 内置自举二极管，但开源项目（Odrive、VESC、SimpleFOC）**一律外加肖特基**：

| | FD6288 内置 | 外置 SS14 |
|---|---|---|
| 类型 | IC 内部伪二极管 | 金属-半导体肖特基 |
| Vf | ~0.7–1.0V | **~0.4V** |
| 反向恢复 | 有 | **几乎为零** |
| 充电结果 | 电容充到 ~11V | 电容充到 ~11.6V |
| 成本 | "免费" | ¥0.1/颗 |

> 3S 低电量时电压仅 9V。减 1V 内置压降只剩 8V 栅极驱动。外置肖特基省 0.6V，多出的就是安全裕量。

**推荐 SS14**（SMA, 40V, 1A, Vf≈0.4V, ¥0.1）。
阳极接 VB (12V)，阴极接 VS (半桥输出)，与内置并联。3 颗/路。

### 损耗 @ 15A

```
P_cond = (15A)² × 2.6mΩ × 0.5 = 0.29W
P_sw   = 0.5 × 12V × 15A × (13nC/1.5A) × 20kHz = 0.08W
单管总损耗 ≈ 0.37W，6 颗共 2.2W
```

| 大小 | 5×6mm | 5×6mm | 10×6.5mm |

### 8.6 布板要点

| 要点 | 做法 |
|------|------|
| 功率回路最短 | VBAT→Q1_D→Q1_S→Q2_D→Q2_S→Rsh→GND 闭环面积最小 |
| 栅极回路短 | Rg 贴 MOSFET Gate，HO/VS 走线并行靠近 |
| 底部焊盘 | AON6354 底部大面积铺铜 + 过孔到背面 |
| 自举元件 | 电容和 SS14 紧靠 FD6288 VB/VS |

## 9. 栅极回路设计

### 基本结构

```
FD6288 HO ──┬──[Rg 4.7Ω]──┬── AON6354 Gate
            │              │
            └──►├──────    │  关断二极管 (可选)
                           │
                      [Rgs 10kΩ] ──┐  下拉电阻
                           │        │
                      [Cgs 100pF] ─┘  (可选)
                           │
FD6288 VS ───────────────┴── AON6354 Source
```

### 9.1 栅极电阻 Rg

| Rg | 开关时间 | 效果 |
|----|----------|------|
| 0Ω | ~9ns | 最快，可能振铃 |
| **4.7Ω** | ~15ns | 微阻尼，推荐 |
| 10Ω | ~25ns | 偏保守 |

AON6354 Qg=13nC，FD6288 1.5A 驱动，4.7Ω 防轻微振铃同时几乎不影响速度。

### 9.2 并联二极管 —— 关断加速

```
         ┌──[Rg]──┐
驱动HO ──┤         ├── Gate     导通: 走电阻, 速度可控, EMI低
         └──►├────┘             关断: 走二极管, 绕过电阻, 快速拉低
```

开管可以慢一点（控 EMI），关管必须快——对面管子马上就要开，这边关慢了就直通。二极管让关断电流直接绕过 Rg，实现"开慢关快"的非对称驱动。

> **本项目是否需要？** AON6354 Qg 仅 13nC，Rg 4.7Ω 开关已足够快（关断 ~15ns），不加也能保证安全。如果布局差、走线长导致寄生电感大，加一颗 **BAT54S (SOT-23, 双肖特基)** 或 **1N4148** 在 Rg 两端做保险即可。板子紧凑时可省略。

### 9.3 栅源电阻 Rgs —— 下拉防浮空

```
Gate ──[10kΩ]── Source
```

驱动上电前 MCU IO 是高阻态，没有 Rgs 时栅极浮空。一点点漏电流或 PCB 耦合噪声就可能让 MOSFET 微导通 → 发热甚至烧管。10kΩ 提供一个确定的低电平路径。

> **必备，不可省略。**

### 9.4 栅源电容 Cgs —— 防寄生导通

```
Gate ──[100pF]── Source
```

与 Rg 组成 RC 低通，滤除高频振铃和 **dv/dt 寄生导通**：

下管关断后，上管开通过的瞬间，半桥输出电压从 GND 跳到 VBAT（几十 V/ns）。这个跳变通过下管的 **米勒电容 (Cgd)** 耦合到下管栅极，产生一个尖峰电压。如果尖峰超过 Vgs(th)（AON6354 约 1.5V），下管会被"弹开"→ 上下管瞬间直通。

Cgs 给这个耦合过来的电荷一个低阻抗路径泄放，把尖峰压在 Vgs(th) 以下。

```
取值:
  太小 (<100pF): 不起作用
  太大 (>1nF):  开关变慢，损耗增加
  推荐: 100–470pF

AON6354 Cgd≈400pF, 推荐 Cgs=220pF 起步。
```

> **建议预留焊盘，不一定焊。** 布板好、走线短的情况下寄生电感小，dv/dt 耦合弱，可能不需要。先不焊，示波器看栅极波形有尖峰再加。

## 14. 常见坑

| 坑 | 现象 | 原因 | 解决 |
|----|------|------|------|
| 直通烧管 | 上电冒烟 | 死区没配 | TIM 死区≥200ns |
| MOSFET 烫 | 1A 就烫 | Vgs 不够 | 9V 以上没问题 |
| 电机抖动 | 咔咔不转 | 一相不导通 | 示波器逐相查 |
| 自举失效 | 大占空比上管不导通 | 电容放光 | 占空比<95% |
| DFN 虚焊 | 时通时不通 | 底部焊盘没焊好 | 热风枪多吹几秒/加热台 240°C |
