# 双路无刷 FOC 驱动板模块清单

## 目标电机（×2）
- **型号:** LD2808-V1.9 (PVW1)
- **类型:** 三相无刷外转子电机，**无霍尔传感器、无编码器** → 需无感 FOC
- **KV:** 700KV | **电压:** 3S–4S LiPo (11.1V–16.8V) | **电流:** 约 10–20A
- **重量:** 57g | **轴径:** 3.6mm

---

## 架构总览

```
                ┌─────────────────────────────────────────┐
                │            STM32G431RBT6 (LQFP-64)       │
                │                                          │
  主控板 <──CAN──┤ CAN_RX/TX                               │
  调试器 <──UART─┤ USART1_TX/RX (外接串口)                  │
  编码器 <──SPI──┤ SPI1 (AS5047P)                          │
  电机A  <──TIM1─┤ CH1/1N CH2/2N CH3/3N  (6路PWM)         │
  电机B  <──TIM8─┤ CH1/1N CH2/2N CH3/3N  (6路PWM)         │
  电流A  ──ADC───┤ ADC1/2 IN1, IN2                         │
  电流B  ──ADC───┤ ADC1/2 IN3, IN4                         │
  母线A/B──ADC───┤ ADC1/2 IN5, IN6                         │
                └─────────────────────────────────────────┘
```

## 模块详细

### 1. MCU — STM32G431RBT6 (LQFP-64)

| 资源 | 数量 | 用途 |
|------|------|------|
| TIM1 (高级定时器) | 1 | 电机 A 三路互补 PWM（6通道） |
| TIM8 (高级定时器) | 1 | 电机 B 三路互补 PWM（6通道） |
| ADC1 + ADC2 | 2 | 两路电机 电流 + 母线电压采样 |
| CORDIC | 1 | Clark/Park 变换、三角函数硬件加速 |
| OPAMP (内置) | 4 | 可选：替代 2+2 路电流采样外置运放 |
| SPI1 | 1 | 磁编码器接口 |
| USART1 / LPUART1 | 1 | 外接串口调试 |
| FDCAN | 1 | 与主控板通信 |
| GPIO | 52 | 使能、故障信号、LED 等 |

**选型理由：** G431RBT6 比 G431CBU6（UFQFPN-48）多出 LQFP-64 封装，引脚充裕，便于引出双路 PWM + 通信接口，且 LQFP 更易手焊。

### 2. 引脚分配建议

#### 电机 A — TIM1 互补 PWM
| 信号 | 引脚 | 说明 |
|------|------|------|
| A_H | PE9  (TIM1_CH1) | A 相上管 |
| A_L | PE8  (TIM1_CH1N) | A 相下管 |
| B_H | PE11 (TIM1_CH2) | B 相上管 |
| B_L | PE10 (TIM1_CH2N) | B 相下管 |
| C_H | PE13 (TIM1_CH3) | C 相上管 |
| C_L | PE12 (TIM1_CH3N) | C 相下管 |
| BKIN | PE15 (TIM1_BKIN) | 刹车输入（过流保护） |

#### 电机 B — TIM8 互补 PWM
| 信号 | 引脚 | 说明 |
|------|------|------|
| A_H | PC6  (TIM8_CH1) | A 相上管 |
| A_L | PA7  (TIM8_CH1N) | A 相下管 |
| B_H | PC7  (TIM8_CH2) | B 相上管 |
| B_L | PB0  (TIM8_CH2N) | B 相下管 |
| C_H | PC8  (TIM8_CH3) | C 相上管 |
| C_L | PB1  (TIM8_CH3N) | C 相下管 |
| BKIN | PA6  (TIM8_BKIN) | 刹车输入 |

#### 电流采样 — ADC
| 信号 | ADC 通道 | 引脚 | 说明 |
|------|----------|------|------|
| IA_A | ADC1_IN1 | PA0 | 电机 A A相电流 |
| IB_A | ADC1_IN2 | PA1 | 电机 A B相电流 |
| IA_B | ADC2_IN1 | PA4 | 电机 B A相电流 |
| IB_B | ADC2_IN4 | PA5 | 电机 B B相电流 |
| VBUS_A | ADC1_IN3 | PA2 | 电机 A 母线电压 |
| VBUS_B | ADC1_IN4 | PA3 | 电机 B 母线电压 |

> 电流采样使用 ADC1+ADC2 双 ADC 同步注入模式，两路电流可同时采样消除 PWM 噪声。

#### 通信接口
| 功能 | 外设 | 引脚 | 说明 |
|------|------|------|------|
| 调试串口 | USART1 | PA9(TX), PA10(RX) | 外接 CH340/CP2102 USB-TTL 模块 |
| 主控板通信 | FDCAN | PB8(RX), PB9(TX) | 需外接 CAN 收发器（SN65HVD230） |
| 磁编码器 | SPI1 | PA5(SCK), PA6(MISO), PA7(MOSI), PB2(CS) | AS5047P / MT6701 |
| SWD 调试 | SWDIO/SWCLK | PA13, PA14 | ST-Link 编程口 |

### 3. 三相逆变桥 ×2（12×N-MOSFET）
- 每路 6 颗 N 沟道 MOSFET，两路共 12 颗
- **选型要点:**
  - Vds > 30V
  - Rds(on) < 10mΩ（20A 单管损耗 < 2W）
  - 封装: SO-8 双管可省 PCB 面积
- **典型选型:** FDMS7658AS、SIR8724、TPN2R608

### 4. 栅极驱动 ×2（双路 Gate Driver）
每路三相半桥需独立栅极驱动：
- **FD6288 ×2**（每路一颗，三路半桥，自举，最常用）
- 或 **MP6530 ×2**（集成预驱，更小体积）
- 或 **IR2101S ×6**（每路 3 颗，可独立检修）

### 5. 电流采样（2-Shunt ×2）
> 每路电机 2 电阻（A/B 相），两路共 4 颗采样电阻。

- **方案 A — 外置运放:** INA240A1 ×4（PWM 抑制能力强）
- **方案 B — 内置 OPAMP:** G431 内置 4 个运放刚好覆盖 2+2 电流采样
- **采样电阻:** 5–20mΩ / 1W / 2512

### 6. 母线电压采样 ×2
- 每路独立分压采样，共 2 路
- 分压比: 20Vmax → 约 3.0V（例如 1:6.67）

### 7. 电源树

| 电压 | 用途 | 方案 |
|------|------|------|
| VBAT (11.1–16.8V) | 双路电机电源 | 锂电池直入 |
| 12V | 栅极驱动供电 ×2 | 降压/升压 DC-DC（如 TPS54331） |
| 3.3V | MCU + 编码器 + CAN收发器 | VBAT → DC-DC 降压（如 MP1584） |

> 双路电机运行时总电流可达 40A，电源入口走线/端子需足够承载。

### 8. 保护电路
- **过流保护:** TIM1/TIM8 BKIN 刹车输入，栅极驱动 ITRIP 或比较器触发
- **母线电压监控:** 各相 ADC 采样，软件保
- **温度保护:** NTC 贴 MOSFET 旁，接入 ADC 通道（如 PC4/PC5）

### 9. 磁编码器（有感 FOC，用于调试基准）
- **型号推荐:** AS5047P（14bit SPI）或 MT6701（性价比）
- **接口:** SPI1，SCK / MISO / MOSI / CS（见 §2 引脚表）
- **用途:** 调试阶段先上编码器校准电流环，无感观测器完成后再去除/做对比

### 10. 主控板通信 — CAN
- **协议:** CAN 2.0B，推荐 CANOpen 或自定义轻量帧
- **收发器:** SN65HVD230（3.3V 供电，直连 G431）
- **帧设计建议:**
  - 主控 → 驱动板: 目标转速/扭矩、使能、PID 参数
  - 驱动板 → 主控: 实际转速、电流、故障状态
- **波特率:** 500K / 1Mbps

### 11. 调试串口 — USART1
- 外接 USB-TTL 模块（CH340 / CP2102），3 线: TX / RX / GND
- 输出实时波形数据至 Vofa+ / SerialStudio 上位机
- 波特率: 115200 或更高

## 注意事项
1. 两路大电流需独立功率回路，地线汇聚到电源入口单点接地。
2. **ADC 采样时序必须与 PWM 中心对齐**，在 PWM 中点触发 ADC 转换。
3. 栅极驱动走线短且远离模拟区域，避免振铃/EMI。
4. 初次上电前用限流电源（1A），先调一路电机，确认波形正常再上第二路。
5. **CAN 终端电阻 120Ω** 需在板上预留跳线位置。
