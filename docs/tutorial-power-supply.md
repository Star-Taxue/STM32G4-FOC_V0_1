# 电源与接地 — 从原理到布板

>  写给刚接触硬件设计的新手。先从"地到底是什么"讲起，再到本项目电源架构、地划分、布板要点。

---

##  Part A — 接地：最重要的基础概念

###  A1. "地" 到底是什么

先说透一个反直觉的事实：**"地"不是零电压的魔法平面——它只是一条回流导线。**

电流必须形成回路才能流动：

```
 电池+ ──→ 负载（电机/MCU/运放）──→ 电池-
   ↑                                  │
   └──────────── 回流路径 ────────────┘
```

这条回流路径就是我们叫"地"的铜。它和正极走线一样，有电阻、有电感、会发热。

**"地不是零"的例子：**

```
10A 电流流经 10mm 长、1mm 宽的 1oz 铜皮（约 5mΩ）:

  V = 10A × 0.005Ω = 50mV

  这段"地"的两端有 50mV 的压差。
  如果你在这段地的两端各接一根线，它们不是同一个电位。
```

50mV 对于 10A 的功率电路无所谓，但对于一个测 ±32mV 差分信号的电流运放来说就是灾难——**"地"上的 50mV 噪声直接等于输入信号的 150% 误差。**

###  A2. 为什么不同的地要分开

电路中不同部分对"地"的干净程度要求完全不同：

| 电路类型 | 电流 | di/dt | 对地噪声容忍度 |
|----------|:----:|:-----:|:----:|
| 功率回路（MOSFET + 电机） | 10–20A | 巨大 | 极高（噪声就是信号） |
| 数字电路（MCU） | 几十mA | 中等 | 中等（有个阈值，超了才出错） |
| 模拟电路（运放） | 微安级 | 极小 | **极低（μV 级干扰就能毁掉精度）** |

**如果在同一块铜皮上混这些信号：**

```
 功率地铜皮 ──────────────────────────
   ↑      ↑      ↑      ↑      ↑
  MOSFET MOSFet  MCU   运放  运放
 (10A)  (10A)  (50mA) (μA)  (μA)

 MOSFET 开关瞬间，10A 电流在 ns 级关断：
 V_noise = L_parasitic × di/dt = 5nH × (10A/10ns) = 5V !!

 这 5V 的噪声脉冲出现在整条铜皮上 → 运放、MCU 全看到
 → 运放输出乱跳 → ADC 数据全是噪声 → FOC 崩溃
```

**解决思路：** 给"脏"电流和"干净"电路各走各的回流路径，只在唯一的地方汇合。

###  A3. 星形接地 (Star Ground)

```
           ┌─────────┐
           │  总接地点 │  ← 电池负极入口，唯一汇合点
           └─┬──┬──┬──┘
             │  │  │
    ┌────────┘  │  └────────┐
    ▼           ▼           ▼
 ┌──────┐  ┌──────┐   ┌──────┐
 │功率地│  │数字地│   │模拟地│
 │(粗铜)│  │(MCU) │   │(运放)│
 └──┬───┘  └──────┘   └──────┘
    │
 ┌──┴──┬──────┬──────┐
MOSFET MOSFET MOSFET
 (10A)  (10A)  (10A)

 三条地分开走，各回各的，只在电池负极入口碰面。
 功率地上的 5V 噪声不会传导到模拟地。
```

**物理实现：** 在 PCB 上，这三块地是**物理分离的铜皮**，只在电池负极连接器的焊盘处用一根走线或过孔连到一起。

###  A4. 常见的错误认知（很重要！）

| 错误认知 | 实际情况 |
|----------|----------|
| "地都是连在一起的，分开没用" | 虽然最终连通，但**噪声电流走的路径不同**。分开的意义是**控制噪声电流流向** |
| "多加过孔把地连好就行" | 过孔多能降低 DC 电阻，但不能阻止噪声——噪声走**阻抗最低**的路径，通常是最短/最宽的 |
| "用磁珠隔离数字地和模拟地" | 可以，但磁珠在高频时呈现高阻抗 → 两块地在高频时被隔离，低频时连通 |
| "铺一整块地平面就行" | 整块地平面对低频是好的回路，但对高频开关噪声，它会把噪声均匀散布到全板 |

>  **核心思想：星形接地不是"把地切开"，而是"控制噪声电流只能在自己那一块地上流动，不能跑到别人的地上"。**

##  Part B — 电源架构：你需要几路电

###  B1. 需求分析

| 用电设备 | 电压要求 | 电流需求 | 对电源噪声敏感？ |
|----------|----------|:--------:|:---:|
| FD6288 栅极驱动 ×2 | 7–25V | ~20mA 静态 + 峰值栅极充电 | 否（本身是大噪声源） |
| MCU (STM32G431) | 1.71–3.6V | ~30mA | 是（内部 ADC 参考） |
| INA240A2 ×4 | 2.7–5.5V | 2.4mA ×4 ≈ 10mA | **极是**（模拟信号链） |
| 磁编码器 (AS5047P) | 3.3V / 5V | ~15mA | 中等 |
| CAN 收发器 (SN65HVD230) | 3.3V | ~15mA | 中等 |

**总结：需要两路电压**
- **栅极驱动电压**：7–25V（FD6288）
- **低压电**：3.3V（MCU + 运放 + 编码器 + CAN）

###  B2. 栅极驱动可以直接用电池吗

FD6288 的供电范围 7–25V，3S 电池 9–12.6V、4S 电池 12–16.8V——全在范围内。

**方案 A — 电池直供 FD6288：**

```
 电池+ ──→ 电机功率回路
    │
    └──→ FD6288 VB (经 LC 滤波)
```

- 优点：零额外元件，最简单
- 缺点：栅极驱动电压随电池波动。3S 低电时 Vgs 仅 ~8V（减自举压降），AON6354 的 Rds(on) 略增（约 +15%），但仍在安全范围内

**方案 B — 独立 10V 稳压供 FD6288：**

```
 电池+ ──→ DC-DC Buck → 10V → FD6288 VB
```

- 优点：栅极驱动电压恒定，与电池电压解耦
- 缺点：多一路 DC-DC，多占地、多花钱

**推荐方案 A（电池直供），理由：**
1. FD6288 接受宽压，AON6354 在 Vgs=8V 时仍可靠工作
2. 少一路 DC-DC = 少一个噪声源、少一个故障点
3. 新手友好——元件越少，犯错机会越少

> 如果想加保险，在 VBAT 到 FD6288 VB 之间串一个 **LC 滤波**（10μH 电感 + 100μF 电解），滤掉 MOSFET 开关在电池线上产生的高频噪声。

###  B3. 3.3V 怎么来 —— 降压方案选型

电池 9–16.8V → 3.3V。两种方式：

| | Buck（DC-DC 开关） | LDO（线性稳压） |
|---|---|---|
| 原理 | 开关管高频通断 + 电感储能 | 像一个自动可调电阻 |
| 效率 | 85–95% | 16.8→3.3: η=3.3/16.8=**19.6%** |
| 发热 @100mA | 几乎不热 | (16.8-3.3)×0.1 = **1.35W 要散热** |
| 噪声 | 有开关纹波（可滤） | 极低 |
| 适合 | 压差大的场合 | 压差小（<2V）的场合 |

**结论：16.8V → 3.3V，压差 13.5V，必须用 Buck。** 如果用 LDO，不仅要烧 1W+ 的热量，而且 3S 电池 12.6V 时效率也只有 26%——电池续航直接废。

###  B4. Buck 芯片推荐 — MP9942

| 参数 | 要求 | MP9942 | 说明 |
|------|------|--------|------|
| 输入电压 | 9–16.8V | 4–30V | 裕量充裕 |
| 输出电压 | 3.3V | 可调（外接反馈电阻） | |
| 输出电流 | ~100mA | **2A** | 裕量 20×，完全不热 |
| 开关频率 | — | 410kHz | 远离 FOC 20kHz，不会互调 |
| 封装 | — | SOT-23-8 | 手焊友好 |
| 同步整流 | — | **内置** | 无需外接续流二极管 |
| 价格 | — | ~¥3 | |

> 备选：**MP1584**（SOP-8，老牌经典，更大量现货）、**TPS54331**（TI 高端，但 ¥8 偏贵）。本项目用 MP9942（引脚少、内置同步管、频率固定）。

###  B5. 3.3V 后为什么还要滤波

Buck 输出的是 3.3V 直流 + 开关纹波（几十 mV，几百 kHz）。MCU 数字部分对这个纹波不太敏感，但 INA240 的电源抑制比有限：

```
Buck 输出 ──→ [LC 滤波] ──→ 干净 3.3V → INA240
                          │
                          └──→ MCU 数字电源
```

**手段：** 在 Buck 输出和模拟 3.3V 之间串一个 **磁珠（Ferrite Bead）**：
- DC 时：几乎零电阻（<0.1Ω）
- 100MHz 时：几百 Ω 阻抗，强力衰减开关噪声

```
 Buck OUT ──[磁珠 600Ω@100MHz]──┬── 模拟3.3V (INA240, ADC参考)
                                │
                               [10μF + 100nF] → GND
```

**磁珠型号推荐：** BLM18PG121SN1（0603，120Ω@100MHz）或等同规格。

>  磁珠和电感的区别：磁珠把高频能量变成热量耗散掉，电感把能量存起来反弹回去。滤电源噪声用磁珠——它只吃掉不反弹。

##  Part C — 去耦电容：每个芯片的"本地水库"

###  C1. 为什么每个芯片都需要自己的电容

芯片（尤其是 MCU 和运放）内部的晶体管在纳秒级开关，需要瞬态电流。PCB 走线有电感，电感抗拒电流突变：

```
 电源 ────[走线电感 L]──── 芯片电源引脚

 芯片瞬间需要 10mA，走线电感不让电流突变 →
 芯片电源引脚电压瞬间跌落 → 逻辑出错/模拟偏差
```

**去耦电容就是放在芯片电源脚旁边的一个"本地水库"：**

```
 电源 ────[走线电感 L]──┬── 芯片电源引脚
                        │
                      [100nF]
                        │
                       GND

 当芯片瞬间需要电流时，电容在纳秒内放电供应，
 走线上的电流慢慢补回来。
```

###  C2. 去耦电容的选值与放置

| 电容 | 值 | 封装 | 放哪 |
|------|-----|------|------|
| 高频去耦 | 100nF / X7R | 0603 | **每个芯片的电源引脚旁边，越近越好（<3mm）** |
| 大容量储能 | 10μF / X7R | 0805 | 每 3–5 个芯片放一个，靠近电源入口 |

**放置规则：**

```
  电源走线 ──→ [电容] ──→ 芯片电源脚

  必须是 电源 → 电容 → 芯片 的顺序！
  电容要"挡在"电源和芯片之间。

  错误：电源 → 芯片 → 电容（电容变摆设）
```

**实物距离：** 100nF 电容和芯片电源引脚的距离**不超过 3mm**（越近越好）。3mm 听起来很苛刻，但在 0603 封装的 PCB 上很容易做到——直接把电容放在芯片背面（Bottom 层），正对电源脚。

##  Part D — 本项目的地划分方案

###  D1. 总体策略

```
                        电池负极 (唯一汇合点 — Star Point)
                              │
              ┌───────────────┼───────────────┐
              │               │               │
         ┌────┴────┐    ┌────┴────┐    ┌─────┴─────┐
         │ 功率地   │    │ 数字地   │    │  模拟地    │
         │ PGND    │    │ DGND    │    │  AGND     │
         └────┬────┘    └────┬────┘    └─────┬─────┘
              │               │               │
    ┌─────────┼──────┐       MCU          INA240×4
    │         │      │       GND          运放电路
  电机A   电机B    栅极驱动              （采样、滤波）
(采样电阻 (采样电阻  FD6288
 功率地)  功率地)   GND
```

###  D2. 谁属于哪块地

| 元件/区域 | 归哪块地 | 原因 |
|-----------|:---:|------|
| 三相逆变桥 MOSFET 功率回路 | **PGND** | 20A 大电流，噪声最大 |
| 采样电阻的"功率焊盘" | **PGND** | 电流走这里 → 地 |
| 采样电阻的 Kelvin 感线 IN- | **AGND** | 这是给运放的信号线，0 电流 |
| FD6288 电源地 | **PGND** | 栅极驱动峰值电流 1.5A |
| MCU 地 | **DGND** | 数字噪声（时钟、IO 切换） |
| INA240 GND 引脚 | **AGND** | 模拟运放，最怕噪声 |
| INA240 REF2 引脚 | **AGND** | 设定输出零点的基准 |
| 电压分压电阻 R2 | **AGND** | 模拟电压信号 |
| 编码器、CAN 收发器 | **DGND** | 数字通信，归 MCU 侧 |
| Buck 转换器地 | **PGND** | 开关噪声，归功率侧 |

###  D3. 三块地如何汇合

在 PCB 上，三块地是分离的铜皮，**只在电池负极连接器处汇合**：

```
 ┌─ PCB 俯视图 ────────────────────────────────────────┐
│                                                        │
│   ┌──── 功率区域(PGND) ────┐  ┌── 数字(DGND) ──┐      │
│   │  MOSFET ×12             │  │  MCU            │      │
│   │  FD6288 ×2              │  │  编码器          │      │
│   │  采样电阻功率焊盘        │  │  CAN收发器       │      │
│   │                         │  │                 │      │
│   └─────────────────────────┘  └─────────────────┘      │
│                 │                      │                │
│                 │            ┌─────────┴──────┐         │
│                 │            │ 模拟区域(AGND)  │         │
│                 │            │  INA240 ×4      │         │
│                 │            │  电压采样分压    │         │
│                 │            └────────┬────────┘         │
│                 │                     │                  │
│                 └─────────┬───────────┘                  │
│                           │                              │
│                     ┌─────┴─────┐                        │
│                     │ 电池负极   │  ← 唯一汇合点            │
│                     │ 连接器     │                        │
│                     └───────────┘                        │
└──────────────────────────────────────────────────────────┘
```

**关键：** 汇合点的选择很重要。一定要在**噪声容限最大**的地方——也就是电池负极入口。如果汇合点选错了，噪声电流会先经过干净区域再回到电源，把干净地也弄脏。

###  D4. 不对——那 Buck 转换器的地算哪个

刚才说了 Buck 归 PGND。但 Buck 输出的是给 MCU 和 INA240 用的 3.3V。这两种电是同一个 Buck 出来的，却接了不同的地。

这是电源设计里最容易被忽视的问题：

```
                    ┌── Buck 转换器 ──┐
                    │   (PGND)        │
                    │                 │
 电池+ ─────────────┤ IN          SW  ├──[电感]──┬── 3.3V输出
                    │                 │          │
                    │   GND ──────────┼──────────┼──── 
                    └─────────────────┘          │
                                                 │
            ┌────────────────────────────────────┘
            ▼
    ┌──────────────┐
    │  LC 滤波 +   │
    │  磁珠隔离     │
    └──────┬───────┘
           │
    模拟 3.3V ──── INA240 (AGND)
    数字 3.3V ──── MCU    (DGND)
```

**处理方式：** Buck 的输出经过磁珠后，地参考**切换**——Buck 侧的地是 PGND，磁珠后的模拟 3.3V 参考 AGND，数字 3.3V 参考 DGND。三块地在电池负极汇合。

实际上电路是这样连的：

```
 Buck OUT ──[磁珠1]── 模拟3.3V ─────→ INA240 VS
          │                          INA240 GND → AGND
          │
          └─[磁珠2]── 数字3.3V ─────→ MCU VDD
                                     MCU VSS  → DGND

 PGND ──────────────────────┐
 AGND ──────────────────────┼── 电池负极（汇合点）
 DGND ──────────────────────┘
```

###  D5. 如果你用整块地平面（不分地）会怎样

这是新手最常见的做法——看到每个板子都有一整块铜皮当地，就仿照着铺一块完整的地平面。

**对于纯数字电路（如 Arduino 板、Raspberry Pi）：** 整块地平面确实是最佳方案——提供最低阻抗回流路径，没有模拟信号的精度要求。

**对于本项目的混合信号电路（20A 功率 + 微伏级模拟）：** 整块地平面等于把所有噪声均匀涂满整块板子。两种电路会互相影响，最严重的情况是：

1. MOSFET 开关噪声通过地平面耦合到 INA240 输入 → 运放输出尖峰
2. MCU 数字噪声耦合到 ADC 参考 → ADC 读值抖动
3. 如果地平面不够厚（1oz），大电流路径上的压降会让**不同位置的"地"电位不同**——运放的 IN- 和 REF2 参考的不是同一个地电平

**结论：本项目必须分地。** 不分地不一定会失败，但它把"调试难度"从"示波器看一眼就能定位"变成了"抓狂两天找不到原因"。

##  Part E — 电源板方案：独立供电板

###  E1. 为什么要独立做电源板

| 一体的利弊 | 分离的利弊 |
|------------|------------|
| 一块板子搞定，紧凑 | 两块板子，多一对连接器 |
| 功率噪声和模拟电路在同一块 PCB | **功率地和信号地在物理上分离，天然隔离** |
| 调试时改电源要动整块板 | 电源板单独调试，坏了只换小电源板 |
| — | 可复用——以后做别的电机驱动板还能用 |

**推荐分离。** 对新手尤其友好——电源板简单，先调通电源板，确认各路电压出来是干净的，再接主板。

               │     │   ┌─[保险丝15A]─┐   │
               │     ├───┤              │   │
               │     │   └──────────────┘   │
               │     │         │            │
               │     │    ┌────┴───────┐    │
               │     │    │  MP9942    │    │
               │     │    │  Buck      │    │
               │     │    └────┬───────┘    │
               │     │         │            │
               │     │    ┌────┴───────┐    │
               │     │    │ LC滤波+磁珠 │    │
               │     │    └──┬─────┬───┘    │
               │     │       │     │        │
               │     │  3.3V_A  3.3V_D      │
               │     │  (模拟)  (数字)       │
               │     │       │     │        │
               │     ├───────┴─────┴── GND  │
               │     │                     │
               │     │  VBAT直出 → 主板     │
               │     │                     │
               └─────┼─────────────────────┘
                     │
               主板接口 (排针/排母)
```

###  E3. 板间连接器

电源板到主板至少需要：

| 信号 | 数量 | 电流 | 连接器选型 |
|------|:---:|------|------|
| VBAT (电池直通) | 1 | 40A 峰值 | XT60 走线 / 粗硅胶线直焊 |
| 3.3V_A (模拟电源) | 1 | ~15mA | 排针 2.54mm |
| 3.3V_D (数字电源) | 1 | ~60mA | 排针 2.54mm |
| AGND | 1 | — | 排针 2.54mm |
| DGND | 1 | — | 排针 2.54mm |
| PGND | 1 | 40A | 粗线直焊或大电流端子 |

**VBAT 和 PGND 大电流流过连接器 → 建议直接焊接粗线**（14AWG 硅胶线），不要过排针。其他信号电流小，排针/排母完全够用。

###  E4. 连接方式建议

```
 电池 (XT60)
   │
   ├── 粗红 ──→ 主板 VBAT 功率端子（焊盘/螺丝端子）
   ├── 粗黑 ──→ 主板 PGND 功率端子
   │
   └── 细线 ──→ 电源板 输入
                │
                └── 排针输出 ──→ 主板
                    Pin1: VBAT (经保险丝)
                    Pin2: 3.3V_A
                    Pin3: 3.3V_D
                    Pin4: AGND
                    Pin5: DGND
                    Pin6: PGND
```

##  Part F — Buck 电路设计 (MP9942)

###  F1. 完整的 3.3V Buck 电路

```
                    MP9942 (SOT-23-8)
                       ┌──┴──┐
 VBAT ──[10μF]──┬──────┤ VIN  │
  9-16.8V       │      │      │
               [100nF] │      ├── SW ──[L 10μH]──┬────── 3.3V OUT
                │      │      │                  │
                │  ┌───┤ EN   │                  ├──[Rfbt 52.3k]──┬── FB
                │  │   │      │                  │                │
    PGND ───────┴──┴───┤ GND  │                  │               [Rfbb 10k]
                       │      │                  │                │
                       │      ├── BST ──[100nF]──┤                │
                       └──────┘       ┌─ SW      │                │
                                      │          │              PGND
                                      │     ┌────┴────┐
                                      │     │ 输出电容  │
                                      │     │ 22μF    │
                                      │     │ [100nF] │
                                      │     └────┬────┘
                                      │          │
                                    PGND ───────┴────── PGND
```

###  F2. 关键元件选型

| 元件 | 值 | 封装 | 说明 |
|------|-----|------|------|
| 输入电容 Cin1 | 10μF / 50V / X7R | 1206 | 稳定输入电压，吸收 Buck 输入电流脉冲 |
| 输入电容 Cin2 | 100nF / 50V / X7R | 0603 | 高频去耦 |
| 电感 L | 10μH / 饱和 > 1A | CD54 贴片功率电感 | 储能元件，值越大纹波越小 |
| 升压电容 Cbst | 100nF / 25V / X7R | 0603 | 为上管栅极提供浮动驱动电压 |
| 输出电容 Cout1 | 22μF / 16V / X7R | 1206 | 稳定输出电压 |
| 输出电容 Cout2 | 100nF / 16V / X7R | 0603 | 高频去耦 |
| 反馈电阻 Rfbt | 52.3kΩ / 1% | 0603 | 上分压电阻 |
| 反馈电阻 Rfbb | 10kΩ / 1% | 0603 | 下分压电阻 |

**输出电压设定：**

```
Vout = 0.8V × (1 + Rfbt/Rfbb)
     = 0.8V × (1 + 52.3k/10k)
     = 0.8V × 6.23
     ≈ 4.98V

 不对，我们要 3.3V:
 Rfbt = 10k × (3.3/0.8 - 1) = 10k × 3.125 = 31.2kΩ

 用 31.6kΩ（标准值）→ Vout ≈ 3.33V ✓
```

###  F3. 输出经磁珠分两路

```
 Buck 3.3V OUT ──┬──[磁珠 BLM18PG121]── 3.3V_ANALOG ── INA240×4
                 │                        │
                 │                      [10μF + 100nF] → AGND
                 │
                 └──[磁珠 BLM18PG121]── 3.3V_DIGITAL ── MCU/编码器/CAN
                                          │
                                        [10μF + 100nF] → DGND
```

每个磁珠后面都要有电容——磁珠 + 电容组成 LC 低通，把 Buck 纹波从几十 mV 压到几百 μV。

##  Part G — 布板要点 (电源板)

###  G1. Buck 布板铁律

Buck 转换器的布板是开关电源设计里最重要的环节。一个原理完全相同的电路，布局好效率 93%、纹波 20mV；布局差效率 70%、纹波 200mV、甚至 EMI 超标。

**核心原则：最小化高频电流环路的面积。**

```
Buck 开关导通时的高频电流路径:

 Cin+ ──→ VIN ──→ SW ──→ 电感 ──→ Cout+ ──→ 负载
   ↑                                           │
   └──────────── 回流路径 (GND) ───────────────┘

   ↑                    ↑
   │ 这一段走线面积必须最小！  │
   └──────────────────────────┘
```

**具体做法：**

| 规则 | 做法 |
|------|------|
| 输入电容紧靠 VIN/GND | Cin 放在 MP9942 VIN 和 GND 引脚旁边，<5mm |
| SW 节点面积最小 | SW 到电感的走线尽量短（<10mm），但宽（>1mm）——这个节点电压高速跳变，是最大辐射源 |
| 反馈走线远离 SW | FB 走线远离 SW 和电感，走线要细（10mil），靠近 GND 平面 |
| 输出电容紧靠电感 | Cout 紧靠电感输出端，减少高频环路 |
| 单点接地 | Buck 的 Cin-、Cout-、芯片 GND 在同一块小铜皮上汇合 → 再单线连到 PGND 汇合点 |

###  G2. 电感选择与摆放

电感是 Buck 里体积最大的元件，也是最强的磁场辐射源：

- **选屏蔽电感**（磁路闭合型，如 CD 系列），不要用工字电感（磁路开放的，直接当发射天线）
- 电感远离模拟信号走线（>10mm）
- 如果必须靠近，电感旋转 90° 让磁力线方向和敏感走线垂直

###  G3. 电源板的完整布局示意

```
 ┌──────────── 电源板 PCB ────────────┐
│                                      │
│   XT60  ──→ 保险丝                    │
│  电池入口  [15A]                     │
│    │                                 │
│    │    ┌── Buck 区域(PGND) ──────┐  │
│    │    │  Cin  MP9942  电感  Cout │  │
│    │    │  ██    ██     ██    ██   │  │
│    │    │      最小环路面积         │  │
│    │    └──────────────────────────┘  │
│    │                │                 │
│    │         ┌──────┴──────┐          │
│    │         │  磁珠 + 电容  │          │
│    │         └──┬───────┬──┘          │

###  G4. 主板侧的地接线规范

电源板输出的 AGND 和 DGND 到达主板后，**不要在主板任何地方把它们连起来**。它们只在电源板的星形点汇合。

```
 正确（两根地线独立走到主板，在电源板汇合）:

                    │    主板     │
 电源板 ── 3.3V_A ──┼────→ INA240 VS
        ── AGND   ──┼────→ INA240 GND
        ── 3.3V_D ──┼────→ MCU VDD
        ── DGND   ──┼────→ MCU VSS
                    │            │

 错误（在主板把 AGND 和 DGND 连在一起）:

                    │    主板     │
 电源板 ── 3.3V_A ──┼────→ INA240 VS
        ── AGND   ──┼──┬─→ INA240 GND
        ── DGND   ──┼──┘   ← 这里又多了一个汇合点！
                    │     噪声电流不一定会走回电源板
                    │     可能在主板里绕一圈破坏模拟信号
```

##  Part H — 保护措施

###  H1. 输入保护

| 保护 | 元件 | 位置 | 作用 |
|------|------|------|------|
| 反接保护 | **PMOS 防反接电路** 或 肖特基串联 | 电池入口 | 接反电池不烧板 |
| 过流保护 | **保险丝 15A**（汽车插片式） | 电池入口 | 短路时熔断 |
| 浪涌抑制 | TVS 管 SMBJ20A | 电池入口对地 | 吸收静电/浪涌 |
| 缓启动 | 100μF 电解 + NTC 热敏 | 电池入口 | 上电瞬间电容充电电流不过大 |

###  H2. 防反接电路（必加）

这是新手最容易犯错的地方：XT60 反插，全板瞬间报废。

**方案 A — 串联肖特基：**

```
 电池+ ──→├── 后续电路
        SS54 (5A 40V 肖特基)
 电池- ──────────── GND

 优点：电路极简
 缺点：5A 电流下压降 0.4V → 2W 发热，需要散热
      对 40A 的峰流完全不现实
```

**方案 B — PMOS 防反接（推荐）：**

```
            ┌─ [Rgs 10k] ─┐
            │              │
 电池+ ─────┼── S ─── D ──┼── 系统电源 VBAT
            │   (PMOS)     │
 电池- ─────┼──────────────┼── PGND
            │              │
            └─ [R1 1k] ───┘
                │
            [稳压管 12V]
                │
              PGND

 正确接入: 电池+ → PMOS 体二极管暂时导通 →
           S 极有电压 → Vgs < 0 → PMOS 完全导通 → 压降 = I²×Rds(on)
           20A × 3mΩ = 60mV，几乎零损耗

 反接: Vgs > 0 → PMOS 完全关断 → 电路不通
```

**PMOS 选型：** AOD403（30V, 70A, 4.5mΩ @10V, TO-252）。20A 时压降仅 90mV，功率 1.8W（靠 PCB 铜皮散热足够）。

###  H3. Buck 输出保护

Buck 芯片自身有过流/过热保护，但输出端建议加一个 TVS 管（比如 5V 的），防止输出电压意外飙升（反馈电阻虚焊等情况）烧毁 MCU 和 INA240。

##  Part K — 调试步骤

###  K1. 电源板上电前检查

1. 万用表蜂鸣档：VBAT 和 GND 不短路、3.3V 和 GND 不短路
2. 目检：MP9942 方向对、电容极性对（X7R 无极性，但确认）、二极管/PMOS 方向对
3. 保险丝先**不装**，用稳压电源代替电池

###  K2. 上电调试

1. **限流 100mA，先不给主板供电**（主板不接）
2. 上电，测 MP9942 输入电压 = 电池电压 ✓
3. 测 MP9942 输出 = 3.3V ± 2% ✓
4. 示波器看输出纹波：AC 耦合，< 30mVpp 合格
5. 磁珠后测 3.3V_A 和 3.3V_D：纹波 < 5mVpp ✓
6. 逐步提升限流到 500mA，确认电压不跌落

###  K3. 接主板

1. 先只接 3.3V_D + DGND，通电看 MCU 是否正常启动
2. 再接 3.3V_A + AGND，测 INA240 OUT 零点 (~1.65V)
3. 最后接 VBAT + PGND，限流 1A，开始调电机

##  常见坑

| 现象 | 可能原因 | 排查 |
|------|----------|------|
| Buck 输出电压不对 | 反馈电阻焊错/虚焊 | 万用表量 Rfbt / Rfbb |
| Buck 发烫 | 电感饱和电流不够 | 换更大饱和电流的电感 |
| 3.3V 纹波大 | 输出电容太小或虚焊、电感离输出电容太远 | 示波器看 SW 节点波形 |
| MCU 跑着跑着复位 | 3.3V 瞬态跌落（Buck 瞬态响应不够） | 加大输出电容到 47μF |
| INA240 输出全是噪声 | 磁珠没加或 AGND/DGND 在主板有额外连接 | 主板万用表量 AGND↔DGND 电阻 |
| 接反电池烧板 | 没加防反接 | 加 PMOS 防反接电路 |
