# 单管 vs 双管 MOSFET —— 选哪个？

---

## 1. 什么是单管、什么是双管？

### 单管
一个封装里只有 **一颗** MOSFET，三只脚：Gate / Drain / Source。

```
TO-252 单管               SO-8 单管
  ┌───┐                  ┌──────┐
  │ D │ (大焊盘)         1 ○    ○ 8
  └───┘                 2 ○    ○ 7
  G  S                  3 ○    ○ 6
                        4 ○    ○ 5
```

### 双管
一个封装里集成 **两颗** MOSFET。SO-8 双管最常见，内部连接有两种：

**共漏型 (Common-Drain) — 半桥专用**
```
         D1 (Pin 1)
         │
  Q1 ────┤
         │
         S1 ─── D2 ──── 这就是半个半桥！
                │
         Q2 ────┤
                │
               S2 (Pin 3)
```
> **这正好是一个半桥的上下管结构！** 一个 SO-8 双管 = 一相半桥，三个覆盖 A/B/C 三相。

**共源型 (Common-Source) — 多用于低压大电流并联**
```
         D1     D2
         │      │
  Q1 ────┤      ├─── Q2
         │      │
         S1 ── S2 ── (内部连接)
```

## 2. 直观对比

| | 单管 (TO-252) | 双管 (SO-8 双管) |
|---|---|---|
| **每相需要** | 2 颗 | 1 颗 |
| **三相需要** | 6 颗 | 3 颗 |
| **PCB 面积** | 大（6×TO-252） | 小（3×SO-8） |
| **布线复杂度** | 高 | 低 |
| **手焊难度** | 低（大焊盘） | 中等（需热风枪或熟练拖焊） |
| **散热** | 好（铜皮大焊盘） | 一般（靠 PCB 铺铜） |
| **价格 (单相)** | ~¥5（6×NCE3080K） | ~¥4.5（3×WSD30L40DN） |
| **走大电流** | 好走（粗线连 Drain 焊盘） | 注意 SO-8 引脚宽度 |
| **坏了换** | 单颗换 | 整颗换（一坏管就要丢一整颗） |

## 3. 为什么 SO-8 双管是 FOC 半桥的"标准答案"

看一相的实际布线对比就明白了：

```
     单管方案 (2 × TO-252)             双管方案 (1 × SO-8 双管)
     ┌──────────┐                     ┌──────────┐
     │  上管 Q1  │                     │           │
     │  D──VBAT  │                     │ WSD30L40DN│
     │  S──┬──┐  │                     │           │
     └─────┘  │  │                     │  D1─VBAT  │
              │  │                     │  S1/D2──A │
     ┌─────┐  │  │                     │  S2─GND   │
     │ 下管 Q2  │  ├─── 电机 A 相        │           │
     │  D──┬──┘  │                     └──────────┘
     │  S──GND  │
     └──────────┘

  走线绕来绕去                           一进一出，干净利落
  寄生电感大                             寄生电感小
```

SO-8 双管内部已经把 S1 和 D2 接好了，外部只需接三根线：VBAT → 电机 A相 → GND。这就是半桥走线的最短路径。

**为什么半桥走线要短？**
功率回路的寄生电感在 MOSFET 开关瞬间会产生电压尖峰 `V = L×di/dt`，尖峰叠加到 VBAT 上可能击穿 MOSFET。走线越短电感越小，尖峰越小。

---

## 4. 结论

| 场景 | 建议 |
|------|------|
| **DIY 手焊、第一次搞** | TO-252 单管（NCE3080K）+ 烙铁 |
| **PCB 打样、追求整洁** | SO-8 双管（WSD30L40DN）+ 锡膏/热风枪 |
| **大电流 >30A** | TO-252 单管或 DFN（散热更强） |
| **极致省钱** | SO-8 双管（3颗 vs 6颗，双管更省） |

对于 LD2808（20A 峰值），**两种都行**：
- 洞洞板/万能板 → 选 TO-252 单管（NCE3080K ×6），烙铁即可
- PCB 打样 → 选 SO-8 双管（WSD30L40DN ×3），走线简洁

手焊技巧：SO-8 上锡膏后用热风枪吹，或先在焊盘上拖一层薄锡，放上 IC 后烙铁快速拖焊四边引脚。
