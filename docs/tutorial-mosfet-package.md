# 单管 vs 双管 MOSFET —— 选哪个？

---

## 1. 什么是单管、什么是双管？

### 单管 (Discrete)
一个封装里只有 **一颗** MOSFET。

```
TO-252 单管
  ┌───┐
  │ D │ (大焊盘，PCB 散热)
  └───┘
  G  S

6 颗 TO-252 = 6 颗独立 MOSFET，两两组成一个半桥
```

### 双管 (Dual SO-8)
一个 SO-8 封装里集成 **两颗** MOSFET，内部共漏串联：

```
         D1 (Pin 1) —— VBAT
         │
  Q1 ────┤  上管
         │
         S1 ─── D2 ── 半桥输出 → 电机 A 相
                │
         Q2 ────┤  下管
                │
               S2 (Pin 3) —— GND
```

---

## 2. 关键对比

| | 单管 TO-252 ×6/路 | 双管 SO-8 ×3/路 |
|---|---|---|
| **散热** | **好** —— 背面大片铜焊盘直连 PCB 铺铜 | **较差** —— 两颗管挤在一个 SO-8 内，靠引脚导热 |
| **热耦合** | 上下管独立散热 | 上下管共享一个封装，互相加热 |
| **货源** | 多到数不清，各家都有 TO-252 30V MOS | 共漏双管型号少，特定型号可能断货 |
| **布线** | 稍复杂，功率回路长 | 简洁，三根线搞定一相 |
| **坏了换** | 换一颗几毛钱 | 换一整颗（两根管都没用了） |
| **价格 单路** | ~¥5（6 × ¥0.8） | ~¥4.5（3 × ¥1.5） |

---

## 3. 为什么开源项目几乎都用单管？

Odrive、VESC、SimpleFOC、MESC、ST 官方参考设计……全是两颗分立 TO-252 MOSFET 组成半桥。原因：

### 3.1 散热是硬道理
TO-252 的 Drain 是一整块铜皮焊盘，直接焊在 PCB 大面积铺铜上做热沉。

```
单管 TO-252 散热路径：
  MOSFET 芯片 → 铜框架 → 大铜焊盘 → PCB 2oz 铺铜 → 空气
              ↑____ 截面积大，热阻低

双管 SO-8 散热路径：
  MOSFET 芯片 → 键合线 → Pin 引脚 → PCB 窄铜皮 → 铺铜 → 空气
              ↑____ 截面积小，热阻高
```

**LD2808 跑 20A，单管每颗 ~1W，6 颗共 6W。** SO-8 双管把 2W 挤在一个小封装，温升明显高于两颗 TO-252。

### 3.2 热耦合 —— SO-8 双管的隐忧
双管封装内两颗 MOSFET 距离不到 1mm。上管发热 → 封装整体升温 → 下管 Rds(on) 随温度增加 → 下管更热 → 恶性循环。分立 MOSFET 各自贴在 PCB 不同位置，热独立。

### 3.3 货源和可替换性
随便一家淘宝/LCSC 搜 "30V TO-252 N-MOSFET" 能出来上百个型号，随便换。SO-8 共漏双管就那么几个型号，断货了就得改板。

### 3.4 开源设计的惯性
Odrive 3.x 用了双管，后来 Odrive S1/Pro 全部回退成 TO-252 分立。VESC 从 v4 到 v6 一直是分立。SimpleFOC 官方驱动板也是分立。业界已经达成共识：**分立散热更好，可靠性更高**。

---

## 4. SO-8 双管什么时候用？

| 场景 | 适用 |
|------|------|
| 小电流（持续 <10A 峰值 <15A），空间受限 | **SO-8 双管** |
| 中电流（持续 10–20A），常规尺寸 | TO-252 单管 |
| 大电流（>20A） | TO-252 / TO-263 / DFN5×6 分立 |

---

## 5. 结论：SO-8 双管，但选好点的型号

LD2808 持续 5–10A，SO-8 双管是最佳封装：体积小、布线短、颗数少。

区别在型号：

| | 能用就行 | 好用 |
|---|---|---|
| 型号 | WSD30L40DN | **Si4778DY** |
| Rds(on) | 5.5mΩ | 3.2mΩ |
| 15A 单管损耗 | 0.71W | **0.47W** |
| 单价 | ¥1.5 | ¥3 |
| 3 颗/路 | ¥4.5 | ¥9 |

贵的那 ¥4.5 买来 **34% 更低发热** + **Vishay 品牌可靠性**，值。

追求更极致 → SiZ340DT（PowerPAIR 3×3mm，FoM=31，比 SO-8 还小）。
